Aula 3 - HTML

O que é HTML?

Hypertext Markup Language (Linguagem de Marcação de Hipertexto) é a linguagem mais usada para se criar Websites.

Hipertexto refere-se a maneira que as páginas web ligam-se entre si, por isso as ligações (links) entre elas são chamados de Hyperlinks.

HTML é usado para marcar um documento com tags que os Navegadores (Chrome, Firefox, Safari, Internet Explore, etc) usam para determinar como as páginas web são estruturadas.

Tags

Como foi dito as tags são utilizadas para marcar o conteúdo das páginas, determinar o que é um título, um cabeçalho, um parágrafo, etc. Além disso os navegadores utilizam algumas tags para formatar o texto, deixar a fonte maior ou menor, o texto em negrito ou itálico, etc. Porém não é recomendado usar tags como o meio principal de formatação. Para isso é recomendado utilizar CSS, que estudaremos mais a fundo nas próximas aulas.

Vejamos um exemplo da estrutura básica de um documento HTML:

                
                    <!DOCTYPE html>
                    <html>
    
                        <head>
                            <title>Título da página</title>
                        </head>
                            
                        <body>
                            <h1>Um cabeçalho</h1>
                            <p>Um parágrafo.</p>
                            <p>Outro parágrafo.</p>
                        </body>
                        
                    </html>

A maioria das tags têm uma tag de fechamento correspondente. Como por exemplo a tag <h1> tem a tag de fechamento </h1>. Porém algumas tags não precisam ser fechadas, como por exemplo a tag <!DOCTYPE html>, chamamos essas tags de tags auto-fecháveis.

Para aprender HTML é preciso estudar várias tags e entender como elas se comportam. Por isso veremos algumas tags mais usadas em HTML. No final dessa aula você encontra uma lista de sites que contém mais tags, que podem não ser usadas com tanta frequência mas podem ser úteis em algumas situações.

Elementos HTML

Um elemento HTML é definido por uma tag de abertura. Se o elemento tem um conteúdo ele termina com uma tag de fechamento, semelhante a tag de abertura mas com uma barra "/" na frente do nome do elemento.

No exemplo anterior temos o elemento <!DOCTYPE html> que não possui conteúdo nem tag de fechamento, a tag de abertura é um elemento por si só.

Temos também o elemento: <p>Um parágrafo.</p>. Que é composto por uma tag de abertura <p>, o conteúdo "Um parágrafo" e a tag de fechamneto </p>.

É importante notar que o conteúdo não precisa ser necessariamente apenas texto, pode conter outros elementos, como por exemplo o elemento head, que contém o elemento title.

Parentesco

Existe um conceito de parentesco entre os elementos html. Se um elemento encontra-se "dentro" de outro, ele é chamada de filho (child). Já o elemento de "fora" é chamado de mãe ou pai (parent) do elemento de "dentro". No nosso exemplo o elemento body é pai dos elementos h1 e p, que por sua vez são filhos de body.

Repare que tanto a tag de abertura <h1> quanto a tag de fechamento </h1> de h1 estão dentro de body, algo como o código a seguir não é válido:

                
                    <body>
                        <h1>Um cabeçalho
                    </body>
                        </h1>

A maioria dos navegadores não interpeta o código acima como válido, podendo gerar resultados indesejáveis.

Atributos

As tags de abertura podem ter atributos, que são informações extra que definem caracteristicas dos elementos.

Um atributo é composto de duas partes, o nome e o valor. O nome identifica a propriedade do elemento, e o valor determina como essa propriedade será usada. Vejamos o exemplo abaixo.

                
                    <p id="meu-parágrafo" title="um parágrafo escrito por mim">
                       Olá Mundo!
                    </p>

No exemplo acima o atributo id tem o valor "meu-paragrafo". Esse atributo é usado para identificar um único elemento. Serve para criar links internos na página e é usado em CSS e JavaScript.

Conforme aprendemos outros elementos HTML veremos mais atributos.

!DOCTYPE

No exemplo visto, a primeira tag que aparece é <!DOCTYPE html>. Essa é uma tag usada pelos Navegadores para determinar o tipo do documento, que no nosso cado é do tipo html, isso ajuda os navegadores a mostrarem o documento corretamente.

hmtl

A tag <html> engloba todo o documento html. E contém duas tags filhas principais: head e body.

head

<head> é a tag que representa o cabeçalho do documento, ela contém outras tags que geralmente possuem metadados (dados sobre o próprio documento html) e não o conteúdo (visto pelo usuário) em si.

body

<body> é o corpo do documento html. É nele que se encontra todo conteúdo da página visível pelo usuário.

title

<title> é a tag que contém o título da página. É uma tag filha da tag head. Geralmente esse título é mostrado no topo da janela ou de uma aba do Navegador.

Tags de Heading

As tags <h1> <h2> <h3> <h4> <h5> e <h6> são usadas para criar cabeçalhos no texto da página. A diferença entre os 6 é a importância ou destaque, sendo <h1> o cabeçalho com maior destaque e <h6> o com menor destaque. Importante não confundir com a tag <head> que é o cabeçalho do docomento em si, e não do texto/conteúdo da página.

Parágrafo

A tag <p> é usada para representar um parágrafo. Um texto sempre deve estar entre tags, seja de parágrafo, headings ou outra. Deixar um texto "solto", sem tags em volta, apesar de não causar um erro, é uma má prática, e pode deixar seu documento menos legível para outros desenvolvedores.

pre

Tudo texto que estiver entre as tags <pre> </pre> será mostrado como está escrito, ou seja, respeitando a formatação, espaços, quebra de lnhas, etc.

Tags de Frase

Algumas tags são usadas para das destaque à palavras ou partes de uma frase. Apesar de a maioria deles mudar visualmente o texto entre as tags, não devem ser usados para essa finalidade. A função desses elementos é ajudar os Navegadores e moteres de busca (Google) a entenderem melhor o conteúdo da página. A tabela abaixo mostra algumas dessas tags e suas finalidades.

Tag Finalidade
<em> A palavra é mostrada com ênfase, geralmente em itálico.
<mark> A palavra é marcada com um fundo amarelo.
<strong> A palavra é mostrada como um texto importante, geralmente em negrito.
<abbr> A palavra é uma abreviação, deve conter o atributo title com o texto não abreviado.

Cuidado: existem várias tags e atributos que mudam visualmente os elementos html, porém não é recomendado usá-los com finalidade. O Recomendado é usar CSS que aprenderemos na próxima aula.

Comentários

Comentários são partes do documento que são ignoradas pelo Navegador, são uasados pelos desenvolvedores para escrever lembretes ou explicar o código. Os comentários devem estar entre <!-- e --> vejamos um exemplo:

                
                    <p>
                       Olá Mundo!
                       <!-- Esta parte não será exibida! -->
                    </p>

Imagens

Para exibir uma imagem utilizamos a tag auto-fechável <img />. Existem dois atributos importante para essa tag: src e alt. O atributo src determina o caminho (link) para a imagem, que pode ser um endereço na web ou um caminho relativo ao documento HTML. O atributo alt é um texto alternativo que é exibido caso a imagem não carregue, e também é usado para acessibilidade de pessoas com algum tipo de deficiência visual.

                
                    <img src="https://i.imgur.com/CVarvWs.jpg" alt="Um gato">

Links

A tag <a> é usada para criar um link. Existem dois atributos importante para essa tag: href e taget. O atributo href recebe o link, já target especifica onde o link será aberto, "_self" é usado para abrir o link na própria página, "_blank" é usado para abrir o link em uma nova aba.

                
                    <a href="https://www.google.com" target="_blank">
                        Clique para abrir o Google!
                    </a>

Listas

As tags <ol> e <ul> são usadas para criar listas ordenadas e não ordenadas respectivamente, a diferença é que as listas ordenadas são numeradas. As listas não ordenadas <ul> são mais usadas.

Para criar items em uma lista usamos a tag <li> veja no exemplo a seguir.

                
                    <ul>
                        <li>Item 1</li>
                        <li>Item 2</li>
                        <li>Item 3</li>
                    </ul>

Tabelas

É possível criar tabelas entre as tags <table> </table>. Para definir uma linha na tabela usamos a tag <tr>. Para definir uma célula usamos a tag <td>, ou <th> para uma célula de cabeçalho. As células podem recer os atributos rowspan e colspan que determinam o espaço de quantas céluals ela ocupa na linha e na coluna respectivamente. Veja o exemplo abaixo:

                
                    <table>
                        <tr>
                            <th>Coluna 1</th>
                            <th>Coluna 2</th>
                            <th>Coluna 3</th>
                        </tr>
                        <tr>
                            <td rowspan = "2">Linha 1 Célula 1</td>
                            <td>Linha 1 Célula 2</td><td>Linha 1 Célula 3</td>
                        </tr>
                        <tr>
                            <td>Linha 2 Célula 2</td>
                            <td>Linha 2 Célula 3</td>
                        </tr>
                        <tr>
                            <td colspan = "3">Linha 3 Célula 1</td>
                        </tr>
                    </table>

Formulários

Formulários são muito úteis em websites para que o usuário possa enviar informações. Porém a parte de receber e tratar essas informações é do backend. Com HTML apenas cuidamos da apresentação deste formulário ao usuário.

As tags <form> </form> delimitam um formulário. Entra essas tags podemos colocar várias outras tags que correspondem aos elementos do formulário. Temos por exemplo entrada de texto, caixas de seleção, seleção de arquivos, botões, etc. Estudaremos como enviar os dados de um formulário em outra aula, quando já soubermos um pouco mais sobre backend. Por enquanto vejamos o exemplo abaixo:

                
                     <form>
                        Nome : <input type = "text" name = "nome_usuario" >
                        E-mail: <input type = "email" name = "e-mail" >
                        Senha: <input type = "password" name = "senha" >
                    </form>
