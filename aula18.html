Aula 18 - Transition

Transition é usada para criar uma transição gradual entre valores de certas propriedades CSS. Podemos controlar 4 coisas nas transições: qual propriedade é afetada; o tempo de espera (delay) para a transição começar; o tempo total da transição; e uma função que muda o comportamento da transição.

Para realizar a transição em si precisamos de uma mudança de estado no elemento. Um exemplo de mudança de estado que aprendemos é hover. Assim para realizar uma transição precisamos ter valores diferente de uma propriedade no seletor normal de um elemento e no pseudo-seletor do estado hover. E usar as propriedades de transição para ter o efeito da transição. Veja o exemplo abaixo, sem transição ainda:

No arquivo style.css:

                
                    div {
                        height: 300px;
                        width: 300px;
                        background-color: lightblue;
                        transform: rotate(0deg);
                    }
                    div:hover {
                        transform: rotate(315deg);
                    }

No arquivo index.html:

                
                    <div>Passe o mouse aqui!</div>

Repare que quando colocamos o mouse sobre a div ela gira instantaneamente. Não seria mais agradável se ela girasse aos poucos? É isso que faremos com transition.

transition-property

Usamos transition-property para selecionar a propriedade que queremos aplicar a transição. No nosso exemplo: transition-property: transform; Mas podemos usar isso para diversos outros tipos de propriedades, como por exemplo: color, backgroundcolor, height, width, border, e praticamente qualquer outra propriedade que receba valores numéricos. Para aprender mais sobre essa propriedade clique aqui.

transition-delay

A propriedade transition-delay defina o tempo de espera para a transição começar. Por exemplo se usarmos 2s a transição começará 2 segundos após colocarmos o mouse sobre a div. Para aprender mais sobre essa propriedade clique aqui.

transition-duration

transition-duration define o tempo total que a transição leva para ocorrer completamente, ou seja, o tempo que leva para sair do estado inicial e chegar no estado final. Para aprender mais sobre essa propriedade clique aqui.

transition-timing-function

transition-timing-function define uma função que controla como a transformação ocorre. Por exemplo se ela começa devagar e termina rápida, ou se ela começa rápida, fica devager e termina rápida denovo. Isso ficará mais claro nos próximos exemplos. Para aprender mais sobre essa propriedade clique aqui.

Vamos agora pegar aquele primeiro exemplo que vimos e aplicar todas essas propriedades de transição na nossa div.

No arquivo style.css:

                
                    div {
                        height: 300px;
                        width: 300px;
                        background-color: lightblue;
                        transform: rotate(0deg);
                        transition-property: transform;
                        transition-delay: 1s;
                        transition-duration: 3s;
                        transition-timing-function: ease-in-out;
                    }
                    div:hover {
                        transform: rotate(315deg);
                    }

No arquivo index.html:

                
                    <div>Passe o mouse aqui!</div>

Passe um tempo brincando com os valores, mude-os, experimente novos valores, mude outras propriedades. Use sua criatividade.

transition

A propriedade transition é uma abreviatura para as 4 propriedades de transição que vimos. Com ela podemos usar os 4 valores de uma vez, na ordem: transition-property; transition-duration;transition-timing-function; transition-delay; No exemplo anterior podemos apagar as 4 linhas das propriedades de transição e substituir por apenas uma linha assim: transition: transform 3s ease-in-out 1s; e temos o mesmo resultado.

transition: all 1s;

Já percebemos que as transições são ótimas para dar vida ao nosso site, mas se tivermos vários elementos com mudança de estado (hover ou outros) ou várias propriedades de um elemento mudando, isso pode tomar muito tempo nosso escrevendo as transiçoes. Por sorte temos uma maneira muito usada entre os desenvolvedores frontend para definir transições para todos elementos e todas propriedades ao mesmo tempo. Veja o exemplo a seguir:

No arquivo style.css:

                
                    * {
                        transition: all 1s;
                    }
                    div {
                        height: 300px;
                        width: 300px;
                        transform: rotate(0deg);
                        background-color: lightblue;
                    }
                    div:hover {
                        height: 500px;
                        width: 500px;
                        transform: rotate(315deg);
                        background-color: blue;
                    }

No arquivo index.html:

                
                    <div>Passe o mouse aqui!</div>

Repare em 3 coisas importante aqui:

1 - usamos o seletor universal (*), isso faz com que essas transições valham para todos elementos da página. Adicione novos elementos e mude as propriedades dele no estado hover. Verifique se eles também usam a transição.

2 - usamos apenas 2 valores, o primeiro corresponde a propriedade e o segundo a duração. Omitimos os valores da função e do delay, isso faz com que eles usem seus valores padrão, linear para a função, e 0 para o delay.

3 - no primeiro valor, onde selecionamos a propriedade, usamos "all" isso significa "todas". Ou seja, todas propriedades (transform, width, heigth, background-color, etc...) serão afeadas por essa transição.

Agora você pode brincar à vontade, mudar valores de todas propriedades que quiser, adicionar outras propriedades, outros elementos, seja feliz!
