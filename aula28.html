Nunca faça cálculos com decimais

Devido ao tipo de váriavel que o JavaScript usa para números, podem ocorrer alguns erros bizarros quando fazemos cálculos com números decimais. Veja o exemplo:

                
                    console.log(0.1 + 0.2 === 0.3);
                    console.log(0.1 + 0.2);

Estranho né? Como fazer para evitar isso? Basta multiplicar os valores até ficarem inteiros e depois dividí-los veja:

                
                    let valor1 = 10.2;
                    let valor2 = 3.1;

                    function somarDinheiro(a, b) {
                        let resultado = 100*a + 100*b;
                        resultado /= 100;

                        return resultado;
                    }

                    let soma = somarDinheiro(valor1, valor2);

                    alert(soma);

tofixed()

O método toFixed() é usado para arredondar um número decimal para um certo número de casas decimais que é usado como parâmetro do método, veja o exemplo:

Digite no console:

                
                    let num = 10.123456789;
                    num.toFixed(1);
                    num.toFixed(2);
                    num.toFixed(3);
                    num.toFixed(4);
                    num.toFixed(5);
                    num.toFixed(6);
                    num.toFixed(7);
                    num.toFixed(8);

isFinite() e isNaN()

Vimos em uma aula passada que existem 3 valores de number especiais: NaN, Infinity e -Infinity. Existem duas funções para reconhecer esses casos especiais.

A primeira delas é a função isFinite(), que primeiro converte o valor para um número e retorna true para números finitos, e false para Infinity, -Infinity, NaN.

Digite no console:

                
                    isFinite(10);
                    isFinite("10");
                    isFinite(true);
                    isFinite(Infinity);
                    isFinite(-Infinity);
                    isFinite(NaN);
                    isFinite("olá");

Outra função é isNaN() que primeiro converte o valor para um número e retorna true caso o resultado seja NaN e false caso contrário.

Digite no console:

                
                    isNaN(10);
                    isNaN("10");
                    isNaN(true);
                    isNaN(Infinity);
                    isNaN(-Infinity);
                    isNaN(NaN);
                    isNaN("olá");

parseInt() e parseFloat()

As funções parseInt() e parseFloat() leêm uma string que começa com um número até encontar um outro tipo de caractere e retorna o número lido até então, veja o exemplo:

Digite no console:

                
                    parseInt("10px");
                    parseInt("33.5C");
                    parseInt("10px 20px");
                    parseInt("R$10.00");
                    parseFloat("10px");
                    parseFloat("33.5C");
                    parseFloat("10px 20px");
                    parseFloat("R$10.00");

O objeto Math

Em JavaScript temos um objeto especial Math que possui algumas propriedades e métodos úteis para numbers. Veremos alguns desses métodos, caso queira estudar todas propriedades e métodos do objeto Math, clique aqui.

Math.floor(x)

Esse método é utilizado para arredondar um número para o maior número inteiro menor que x. Veja os exemplos:

Digite no console:

                
                    Matah.floor(10.3);
                    Matah.floor(10.7);
                    Matah.floor(-10.3);
                    Matah.floor(-10.7);

Math.random()

O método Math.random() gera um número aleatório entre 0 (incluindo) e 1 (excluindo). Veja o exemplo:

Digite no console várias vezes:

                
                    Math.random();

O método Math.random() é muito usado em conjunto com o método Math.floor() para criar valores aleatórios inteiros, veja o exemplo abaixo que cria um número aleatório entre 0 e 255:

Digite no console várias vezes:

                
                    let num;
                    num = Math.random();
                    num *= 256;
                    num = Math.floor(num);

Existem outros 3 métodos de arredondamento: Math.ceil(); Math.round(); e Math.trunc(). Procure aprender quais as diferenças entre eles.
