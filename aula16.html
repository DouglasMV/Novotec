Aula 16 - Background

Já conhecemos a propriedade background-color para mudar a cor de fundo de um elemento. Agora vamos aprender como utilizar gradientes e imagens no fundo (background) de um elemento.

Gradient

Também é possível usar gradientes no background de um elemento. Veja o exemplo:

                
                    body{
                        height: 100vh;
                        background: linear-gradient(90deg, purple, violet);
                    }

O primeiro valor é o ângulo de inclinação do gradiente, os outros valores são cores, que podem ser mais do que duas. Mude o valor do ângulo e as cores para ver o que acontece.

Existem outros tipos de gradient, para estudar mais sobre isso clique aqui.

background-image

Podemos usar uma imagem como o fundo de um elemento, utilizando a propriedade background-image. Veja o exemplo:

No arquivo style.css:

                
                    body{
                        background-image: url(https://i.imgur.com/X8ywn07.png);
                    }

background-repeat

Por padrão as imagens de fundo se repetem horizontalmente e verticalmente. Mas podemos mudar isso com a propriedade background-repeat. Se queremos repetir apenas horizontalmente usamos: background-repeat: repeat-x; Se queremos repetir apenas verticalmente usamos: background-repeat: repeat-y; Se não queremos repetir a imagem usamos: background-repeat: no-repeat;

No exemplo anterior adicione a propriedade background-repeat e altere entre os valores citados acima. Veja a diferença entre cada valor.

Múltiplos Backgrounds

É possível utilizar múltiplos backgrounds, várias imagens e até mesmo imagens com uma cor de fundo. No próximo exemplo usamos duas imagens e uma cor de fundo, que só aparece pelo fato das imagens terem fundo transparente. Para isso simplesmente usamos dois valores de url na propriedade background-image, separados por vírgula, assim usamos duas imagens. A propriedade background-repeat também pode receber dois valores, separados por vírgula, caso queira repetir uma imagem e a outra não, por exemplo. Preste atenção na ordem, pois ela é importante, a primeira imagem ficará na frente da segunda. E a cor sempre ficará atrás de todas imagens.

No arquivo style.css:

                
                    body{
                        background-image: url(https://i.imgur.com/jy775z2.png) url(https://i.imgur.com/X8ywn07.png);

                        background-repeat: repeat-x, no-repeat;

                        background-color: lightblue;
                    }

background-size: cover;

Podemos mudar o tamanho da imagem de fundo com a propriedade background-size. Podemos usar dois valores absolutos, um para a largura e outro para a altura. Ou apenas um valor absoluto que será a largura, e a altura se ajustará automaticamente. Mas o mais interessante é o valor cover. esse valor faz a imagem ocupar todo o background. É muito usado para criar fundos bem bonitos. Veja o exemplo:

                
                    body{
                        min-height: 100vh;
                        background-image: url(https://i.imgur.com/Xq1IzlO.jpg);
                        background-repeat: no-repeat;
                        background-size: cover;
                    }

background-attachment: fixed;

A propriedade background-attachment: fixed; faz com que a imagem de fundo não role junto com a página. É um efeito muito usado também em websites modernos. Veja o exemplo:

No arquivo style.css:

                
                    * {
                        margin: 0;
                        padding: 0;
                    }
                    div {
                        height: 100vh;
                        background-color: purple;
                    }
                    #div2 {
                        background-image: url(https://i.imgur.com/Xq1IzlO.jpg);
                        background-repeat: no-repeat;
                        background-size: cover;
                        background-attachment: fixed;
                    }

No arquivo index.html:

                
                    <div></div>
                    <div id="div2"></div>
                    <div></div>

Existem outras propriedades e valores para imagens de background, vimos apenas as principais, mais usadas. Para estudar mais sobre isso, clique aqui.
