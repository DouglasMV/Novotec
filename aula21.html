Aula 21 - Flexbox

O Flexbox permite a criação de layouts responsivos sem o uso de float ou position.

Container (Pai)

Para utilizar Flexbox, primeiro precisamos de um container. O container nada mais é do que um elemento que é pai de todos items. Veja o exemplo:

No arquivo index.html:

                
                    <div id="meu-container">
                        <div class="itens">1</div>
                        <div class="itens">2</div>
                        <div class="itens">3</div>
                    </div>

Agora para que o container se torne um Flexbox, utilizamos a propriedade display, com o valor flex. Assim:

No arquivo style.css:

                
                    .itens {
                        background-color: lightblue;
                        width: 100px;
                        text-align: center;
                        padding: 20px 0;
                        border: 1px solid black;
                    }
                    #meu-container {
                        display: flex; 
                    }

O container pode receber 6 propriedades:

  • flex-direction
  • flex-wrap
  • flex-flow
  • justify-content
  • align-items
  • align-content

Estudaremos cada uma delas agora.

flex-direction (container)

Flexbox organiza os item em apenas uma direção, porém você pode escolher qual delas (horizontal ou vertical), usando a propriedade flex-direction. Ela pode receber 4 valores:

  • row (horizontal da esquerda para a direita, valor padrão)
  • row-reverse (horizontal da direita para a esquerda)
  • column (vertical de cima para baixo)
  • column-reverse (vertical de baixo para cima)

Em inglês row significa linha e column coluna. Então você pode escolher entre mostrar seus itens em uma linha ou em uma coluna. Veja o exemplo:

No arquivo style.css:

                
                    .itens {
                        background-color: lightblue;
                        width: 100px;
                        text-align: center;
                        padding: 20px 0;
                        border: 1px solid black;
                    }
                    #meu-container {
                        display: flex;
                        flex-direction: column-reverse; 
                    }

No arquivo index.html:

                
                    <div id="meu-container">
                        <div class="itens">1</div>
                        <div class="itens">2</div>
                        <div class="itens">3</div>
                    </div>

Mude o valor de flex-direction para os outros valores possíveis e veja os resultados.

flex-wrap (container)

A propriedade flex-wrap determina se os items criam uma nova linha (ou coluna) caso o tamanho deles exceda o tamanho do elemento container ou da tela. Existem dois valores possíveis: wrap que cria uma nova linha caso precise; e nowrap que não cria uma nova linha, e sim diminue os elementos para que todos possam caber na mesma linha, esse é o valor padrão. Veja o exemplo:

No arquivo style.css:

                
                    .itens {
                        background-color: lightblue;
                        width: 100px;
                        text-align: center;
                        padding: 20px 0;
                        border: 1px solid black;
                    }
                    #meu-container {
                        display: flex;
                        flex-wrap: wrap; 
                    }

No arquivo index.html:

                
                    <div id="meu-container">
                        <div class="itens">1</div>
                        <div class="itens">2</div>
                        <div class="itens">3</div>
                        <div class="itens">4</div>
                        <div class="itens">5</div>
                        <div class="itens">6</div>
                        <div class="itens">7</div>
                        <div class="itens">8</div>
                        <div class="itens">9</div>
                    </div>

Mude o valor para nowrap (ou apague a propriedade pois esse é o valor padrão) e veja o que acontece.

flex-flow (container)

A propriedade flex-flow é uma abreviação para as propriedades flex-direction e flex-wrap. ela recebe dois valores, o primeiro é o valor da direção (row, row-reverse, column, column-reverse) e o segundo é se deve ou não criar uma nova linha caso precise (wrap, nowrap). Veja o exemplo:

No arquivo style.css:

                
                    .itens {
                        background-color: lightblue;
                        width: 100px;
                        text-align: center;
                        padding: 20px 0;
                        border: 1px solid black;
                    }
                    #meu-container {
                        display: flex;
                        flex-flow: row-reverse wrap; 
                    }

No arquivo index.html:

                
                    <div id="meu-container">
                        <div class="itens">1</div>
                        <div class="itens">2</div>
                        <div class="itens">3</div>
                        <div class="itens">4</div>
                        <div class="itens">5</div>
                        <div class="itens">6</div>
                        <div class="itens">7</div>
                        <div class="itens">8</div>
                        <div class="itens">9</div>
                    </div>

Repare nesse caso específico o que ocorre quando diminuimos a tela. Mude os valores para ver o que muda.

justify-content (container)

A propriedade justify-content serve para alinhar os itens na direção usada em flex-direction (horizontal para row, vertical para column). Essa propriedade pode receber 5 valores:

  • center (centralizado)
  • flex-start (alinhar ao começo)
  • flex-end (alinha ao final)
  • space-around (cria espaços em volta dos itens)
  • space-between (cria espaço entre os itens)

Veja o exemplo:

No arquivo style.css:

                
                    .itens {
                        background-color: lightblue;
                        width: 100px;
                        text-align: center;
                        padding: 20px 0;
                        border: 1px solid black;
                    }
                    #meu-container {
                        display: flex;
                        flex-direction: row;
                        justify-content: center; 
                    }

No arquivo index.html:

                
                    <div id="meu-container">
                        <div class="itens">1</div>
                        <div class="itens">2</div>
                        <div class="itens">3</div>
                    </div>

Mude o valor para os outros possíveis e veja as diferenças. Mude a direção (flex-direction) também e veja as diferenças.

align-items (container)

A propriedade align-items é usada para alinhar os itens na direção oposta a da propriedade flex-direction. E pode receber 5 valores:

  • center (centralizado)
  • flex-start (no começo do container)
  • flex-end (no final do container)
  • stretch (estica os elementos até preencher a altura, ou largura se a direção for "column", do container)
  • baseline (alinha os itens de acordo com a linha de base do texto.

Veja o exemplo:

No arquivo style.css:

                
                    .itens {
                        background-color: lightblue;
                        width: 100px;
                        text-align: center;
                        padding: 20px 0;
                        border: 1px solid black;
                    }
                    #meu-container {
                        display: flex;
                        background-color: magenta;
                        height: 300px;
                        flex-direction: row;
                        align-items: center; 
                    }

No arquivo index.html:

                
                    <div id="meu-container">
                        <h1 class="itens">1</h1>
                        <h2 class="itens">2</h2>
                        <h3 class="itens">3</h3>
                        <h4 class="itens">4</h4>
                        <h5 class="itens">5</h5>
                        <h6 class="itens">6</h6>
                    </div>

Mude o valor para os outros possíveis e veja as diferenças. Mude a direção (flex-direction) também e veja as diferenças.

align-content (container)

A propriedade align-content é usada para alinhas as linhas (ou colunas se a direção for columns) de um container flexbox. Essa propriedade pode receber 6 valores:

  • center (centralizado)
  • flex-start (alinhar ao começo)
  • flex-end (alinha ao final)
  • space-around (cria espaços em volta dos itens)
  • space-between (cria espaço entre os itens)
  • stretch (estica as linhas até preencher a altura do container)

Veja o exemplo:

No arquivo style.css:

                
                    .itens {
                        background-color: lightblue;
                        width: 100px;
                        text-align: center;
                        padding: 20px 0;
                        border: 1px solid black;
                    }
                    #meu-container {
                        display: flex;
                        flex-direction: row;
                        align-content: center; 
                    }

No arquivo index.html:

                
                    <div id="meu-container">
                        <div class="itens">1</div>
                        <div class="itens">2</div>
                        <div class="itens">3</div>
                        <div class="itens">4</div>
                        <div class="itens">5</div>
                        <div class="itens">6</div>
                        <div class="itens">7</div>
                        <div class="itens">8</div>
                        <div class="itens">9</div>
                    </div>

Mude o valor para os outros possíveis e veja as diferenças. Mude a direção (flex-direction) também e veja as diferenças.

Centralizando perfeitamente

Um problema muito comum é centralizar perfeitamente um elemento dentro de outro. Com Flexbox podemos resolver isso facilmente, veja o exemplo:

No arquivo style.css:

                
                    #meu-container {
                        display: flex;
                        height: 300px;
                        width: 300px;
                        background-color: green;
                        justify-content: center;
                        align-items: center;
                    }

No arquivo index.html:

                
                    <div id="meu-container">
                        <span>Bem no centro!</span>
                    </div>

Itens

Os elementos filhos de um container flex automaticamente se tornam itens flexíveis (flex items).

Esses itens flexíveis podem receber algumas propriedades CSS. Essas propriedades podem ser:

  • order
  • flex-grow
  • flex-shrink
  • flex-basis
  • flex
  • align-self

Vamos estudar cada uma dessas propriedades agora.

order

A propriedade order define a ordem de um item em relação aos outros. O valor deve ser um número, o padrão é 0. Veja o exemplo:

No arquivo style.css:

                
                    .itens {
                        background-color: lightblue;
                        width: 100px;
                        text-align: center;
                        padding: 20px 0;
                        border: 1px solid black;
                    }
                    #meu-container {
                        display: flex;
                    }
                    #div1 {
                        order: 4;
                    }
                    #div2 {
                        order: 1;
                    }
                    #div3 {
                        order: 2;
                    }
                    #div4 {
                        order: 5;
                    }
                    #div5 {
                        order: 3;
                    }

No arquivo index.html:

                
                    <div id="meu-container">
                        <div id="div1" class="itens">1</div>
                        <div id="div2" class="itens">2</div>
                        <div id="div3" class="itens">3</div>
                        <div id="div4" class="itens">4</div>
                        <div id="div5" class="itens">5</div>
                    </div>

flex-grow

A propriedade flex-grow define quanto um item irá crescer em relação aos outros. O valor deve ser um número, o padrão é 0. Só é possível observar o efeito dessa propriedade se o container tive a propriedade align-items: stretch; Veja o exemplo:

No arquivo style.css:

                
                    .itens {
                        background-color: lightblue;
                        width: 100px;
                        text-align: center;
                        padding: 20px 0;
                        border: 1px solid black;
                    }
                    #meu-container {
                        display: flex;
                        align-items: stretch;
                    }
                    #div1 {
                        flex-grow: 1;
                    }
                    #div2 {
                        flex-grow: 1;
                    }
                    #div3 {
                        flex-grow: 4;
                    }

No arquivo index.html:

                
                    <div id="meu-container">
                        <div id="div1" class="itens">1</div>
                        <div id="div2" class="itens">2</div>
                        <div id="div3" class="itens">3</div>
                    </div>

flex-shrink

A propriedade flex-grow define quanto um item irá diminuir em relação aos outros. O valor deve ser um número, o padrão é 0. Só é possível observar o efeito dessa propriedade se o container tive a propriedade align-items: stretch; Veja o exemplo:

No arquivo style.css:

                
                    .itens {
                        background-color: lightblue;
                        width: 300px;
                        text-align: center;
                        padding: 20px 0;
                        border: 1px solid black;
                    }
                    #meu-container {
                        display: flex;
                        align-items: stretch;
                    }
                    #div1 {
                        flex-shrink: 1;
                    }
                    #div2 {
                        flex-shrink: 1;
                    }
                    #div3 {
                        flex-shrink: 2;
                    }

No arquivo index.html:

                
                    <div id="meu-container">
                        <div id="div1" class="itens">1</div>
                        <div id="div2" class="itens">2</div>
                        <div id="div3" class="itens">3</div>
                    </div>

flex-basis

A propriedade flex-basis define a largura inicial de um item. Veja o exemplo:

No arquivo style.css:

                
                    .itens {
                        background-color: lightblue;
                        width: 100px;
                        text-align: center;
                        padding: 20px 0;
                        border: 1px solid black;
                    }
                    #meu-container {
                        display: flex;
                        align-items: stretch;
                    }
                    #div2 {
                        flex-basis:300px
                    }

No arquivo index.html:

                
                    <div id="meu-container">
                        <div id="div1" class="itens">1</div>
                        <div id="div2" class="itens">2</div>
                        <div id="div3" class="itens">3</div>
                    </div>

flex

A propriedade flex é uma abreviação para as propriedades flex-grow, flex-shrink, e flex-basis. Veja o exemplo:

No arquivo style.css:

                
                    .itens {
                        background-color: lightblue;
                        width: 300px;
                        text-align: center;
                        padding: 20px 0;
                        border: 1px solid black;
                    }
                    #meu-container {
                        display: flex;
                        align-items: stretch;
                    }
                    #div2 {
                        flex: 5 2 500px;
                    }

No arquivo index.html:

                
                    <div id="meu-container">
                        <div id="div1" class="itens">1</div>
                        <div id="div2" class="itens">2</div>
                        <div id="div3" class="itens">3</div>
                    </div>

align-self

A propriedade align-self define o alinhamento do item dentro do container flex. Essa propriedade sobrepõe (tem prioridade) a propriedade align-itenms do container. Veja o exemplo:

No arquivo style.css:

                
                    .itens {
                        background-color: lightblue;
                        width: 300px;
                        text-align: center;
                        padding: 20px 0;
                        border: 1px solid black;
                    }
                    #meu-container {
                        height: 300px;
                        background-color: magenta;;
                        display: flex;
                        align-items: flex-start;
                    }
                    #div2 {
                        align-self: center;
                    }
                    #div3 {
                        align-self: flex-end;
                    }

No arquivo index.html:

                
                    <div id="meu-container">
                        <div id="div1" class="itens">1</div>
                        <div id="div2" class="itens">2</div>
                        <div id="div3" class="itens">3</div>
                    </div>
