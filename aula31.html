Aula 31 - DOM

    Nesta aula vamos estudar o Document Object Model (DOM), em português Modelo de Objeto de Documento. Ele fornece uma representação do documento HTML e permite selecionar, mudar e criar elementos HTML. Vamos ver alguns métodos, para uma lista mais completa clique aqui.

    document.getElementById()

    Esse método nos permite selecionar um elemento HTML pelo id e salvar uma referência a ele em uma variável, para depois podermos modificá-lo.

    No index.html:

                
                <div id="meu-id"></div>

    No app.js:

                
                let meuElemento = document.getElementById("meu-id");
                console.log(meuElemento);

    document.querySelectorAll()

    Esse método também serve para selecionar elementos. Ele usa seletores semelhantes ao de CSS e seleciona todos elementos que satisfazem o seletor. Retornando um array com todos elementos selecionados.

    No index.html:

                
                <div class="grid">
                    <div class="item"></div>
                    <div class="item"></div>
                    <div class="item"></div>
                    <div class="item"></div>
                </div>

    No app.js:

                
                let meusElementos = document.querySelectorAll(".grid > .item");
                console.log(meusElementos);

    innerHTML

    Já aprendemos a selecionar um elemento, agora vamos aprender a modificá-lo. A propriedade innerHTML pode ser usada para visualizar ou modificar tudo que se encontra dentro do elemento. Veja o exemplo:

    No index.html:

                
                <h1 id="cabecalho">Olá Mundo!</h1>

    No app.js:

                
                let meuH1 = document.getElementById("cabecalho");
                console.log(meuH1.innerHTML);

    Abra o console e primeiro repare que foi escrito "Olá Mundo!", que é o conteúdo do elemento. Agora escreva no console: 

                
                meuH1.innerHTML = "Oi Universo!";

    Repare que o conteúdo do elemento mudou na página.

    textContent

    A propriedade textContent é semelhante a innerHTML, a diferença é que esta não permite a inserção de código HTML, portanto é mais segura se estamos lidando com algo escrito pelo usuário. Veja a diferênça:

    No index.html:

                
                <h1 id="cabecalho">Olá <span>Mundo</span>!</h1>

    No app.js:

                
                let meuH1 = document.getElementById("cabecalho");
                console.log(meuH1.innerHTML);
                console.log(meuH1.textContent);

    Repare na diferença entre os console.log's. O do innerHTML mostra a tag span, o textContent não. Agora tente modificar o elemento e eja o que acontece em cada caso: 

                
                meuH1.innerHTML = "Oi <span>Universo</span>!";
                meuH1.textContent = "Oi <span>Universo</span>!";

    Atributos

    Elementos HTML possuem atributos, e podemos modificá-los usando JavaScript. Vamos aprender alguns métodos para isso.

    hasAttribute()

    Este método verifica se um elemento possui um atributo. Veja o exemplo:

    No index.html:

                
                <img id="minha-imagem" src="https://i.imgur.com/pycriaD.jpg">

    No app.js:

                
                let imagem = document.getElementById("minha-imagem");
                console.log("Possui o atributo src? " + imagem.hasAttribute("src"));
                console.log("Possui o atributo alt? " + imagem.hasAttribute("alt"));

    getAttribute()

    Este método retorna o valor de um atributo. Veja o exemplo:

    No index.html:

                
                <img id="minha-imagem" src="https://i.imgur.com/pycriaD.jpg">

    No app.js:

                
                let imagem = document.getElementById("minha-imagem");
                console.log("Valor do atributo src: " + imagem.getAttribute("src"));
                console.log("Valor do atributo alt: " + imagem.getAttribute("alt"));

    setAttribute()

    Este método muda o valor de um atributo. Veja o exemplo:

    No index.html:

                
                <img id="minha-imagem" src="https://i.imgur.com/pycriaD.jpg">

    No app.js:

                
                let imagem = document.getElementById("minha-imagem");
                console.log("Valor do atributo alt: " + imagem.getAttribute("alt"));
                imagem.setAttribute("alt", "Jeff Daniels no telefone");
                console.log("Valor do atributo alt: " + imagem.getAttribute("alt"));

    removeAttribute()

    Este método remove o valor de um atributo. Veja o exemplo:

    No index.html:

                
                <img id="minha-imagem" src="https://i.imgur.com/pycriaD.jpg">

    No app.js:

                
                let imagem = document.getElementById("minha-imagem");
                console.log("Valor do atributo src: " + imagem.getAttribute("src"));
                imagem.removeAttribute("src");
                console.log("Valor do atributo src: " + imagem.getAttribute("src"));

    document.createElement()

    Este método cria um novo elemento, com a tag passada como argumento. Veja o exemplo: 

              
              let minhaDiv = document.createElement("div");
              console.log(minhaDiv);

    append()

    Este método serve para colocar um elemento dentro de outro (como filho), no final do elemento pai. Veja o exemplo:

    No index.html:

                
              <div id="container" class="grid">
                <div class="item">Item1</div>
                <div class="item">Item2</div>
                <div class="item">Item3</div>
                <div class="item">Item4</div>
              </div>

    No app.js:

                
                let pai = document.getElementById("container");
                let filho= document.createElement("div");
                filho.textContent = "Item 5!!!";

                pai.append(filho);

    remove()

    Este método remove um elemento. Veja o exemplo:

    No index.html:

                
              <div class="grid">
                <div id="item1" class="item">Item1</div>
                <div id="item2" class="item">Item2</div>
                <div id="item3" class="item">Item3</div>
                <div id="item4" class="item">Item4</div>
              </div>

    No app.js:

                
                let item3 = document.getElementById("item3");

                item3.remove();

    className

    Essa propriedade é uma string que contém o nome de todas classes de um elemento. Veja o exemplo:

    No index.html:

                
              <div id="minha-div" class="vermelho grande">Olá!</div>

    No style.css:

                
              .vermelho {
                color: red;
              }
              .grande {
                font-size: 50px;
              }
              .azul {
                color: blue;
              }
              .pequeno {
                font-size: 10px;
              }

    No app.js:

                
                let minhaDiv = document.getElementById("minha-div");
                console.log(minhaDiv.className);

                //Escreva no console:
                //minhaDiv.className = "azul pequeno";
                //console.log(minhaDiv.className);

    classList

    As vezes queremos remover ou adicionar uma única classe, para isso existe a propriedade classList, que pode ser seguida de 4 métodos que veremos agora:

    classList.contains()

    Este método verifica se o elemento possui uma determinada classe. Veja o exemplo:

    No index.html:

                
              <div id="minha-div" class="vermelho grande">Olá!</div>

    No style.css:

                
              .vermelho {
                color: red;
              }
              .grande {
                font-size: 50px;
              }
              .azul {
                color: blue;
              }
              .pequeno {
                font-size: 10px;
              }

    No app.js:

                
                let minhaDiv = document.getElementById("minha-div");
                console.log("Tem a classe vermelho: " + minhaDiv.classList.contains("vermelho"));
                console.log("Tem a classe azul: " + minhaDiv.classList.contains("azul"));
                console.log("Tem a classe grande: " + minhaDiv.classList.contains("grande"));
                console.log("Tem a classe pequeno: " + minhaDiv.classList.contains("pequeno"));

    classList.add()

    Este método adiciona uma classe a um elemento. Veja o exemplo:

    No index.html:

                
              <div id="minha-div" class="vermelho">Olá!</div>

    No style.css:

                
              .vermelho {
                color: red;
              }
              .grande {
                font-size: 50px;
              }
              .azul {
                color: blue;
              }
              .pequeno {
                font-size: 10px;
              }

    No app.js:

                
                let minhaDiv = document.getElementById("minha-div");
                
                //Digite no console:
                //minhaDiv.classList.add("grande");

    classList.remove()

    Este método remove uma classe a um elemento. Veja o exemplo:

    No index.html:

                
              <div id="minha-div" class="vermelho grande">Olá!</div>

    No style.css:

                
              .vermelho {
                color: red;
              }
              .grande {
                font-size: 50px;
              }
              .azul {
                color: blue;
              }
              .pequeno {
                font-size: 10px;
              }

    No app.js:

                
                let minhaDiv = document.getElementById("minha-div");
                
                //Digite no console:
                //minhaDiv.classList.remove("grande");

    classList.toggle()

    Este método remove uma classe de um elemento se ele possui aquela classe, se ele não possui aquela classe ela é adicionada. Veja o exemplo:

    No index.html:

                
              <div id="minha-div" class="vermelho grande">Olá!</div>

    No style.css:

                
              .vermelho {
                color: red;
              }
              .grande {
                font-size: 50px;
              }
              .azul {
                color: blue;
              }
              .pequeno {
                font-size: 10px;
              }

    No app.js:

                
                let minhaDiv = document.getElementById("minha-div");
                
                //Digite várias vezes no console:
                //minhaDiv.classList.toggle("grande");
