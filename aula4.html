Aula 4 - CSS

O que é CSS?

CSS (Cascading Style Sheets) ou Folhas de Estilo em camadas, é uma linguagem de estilos, feita para facilitar o processo de fazer as páginas de website mais apresentáveis visualmente.

Usando CSS é possível mudar as cores do texto ou de fundo, o estilo e tamanho da fonte, o espaço entre os elementos, o tamanho e posição de cada elemento, entre outras coisas.

CSS dá um controle enorme para o desenvolvedor sobre a apresentação do site. Vamos ver agora como CSS funciona.

Propriedade e Valor

Cada elemento HTML possui várias propriedades que representam algo visualmente naquele elemento, por exemplo a propriedade color representa a cor do texto dentro daquele elemento.

Cada propriedade possui um valor associado, por exemplo podemos associar a propriedade color o valor red, isso faz com que a cor do texto dentro de um elemento seja vermelha.

Para dar um valor a uma propriedade usamos dois pontos (:) entre a propriedade e o valor, e no final usamos ponto e vírgula (;), por exemplo, color: red;

Porém precisamos determinar qual elemento queremos mudar a propriedade, veremos a seguir como fazer isso.

As 3 maneiras de incluir CSS

1 - Inline

A primeira maneira de mudar uma propriedade de um elemento é chamada de Inline, pois mudamos o valor na própria linha em que o elemento HTML se encontra, utilizando o atributo style, veja o exemplo:

                
                    <p style="color: red;">Este texto será vermelho!</p>

Resultado:

Este texto será vermelho!

2 - Embutido

Outra maneira de alterar os estilos CSS é chamada de Embutida. Os estilos são embutidos no documento HTML utilizando-se a tag <style> dentro da tag <head>, o cabeçalho do documento.

Mas para isso precisamos de uma maneira de identificar o elemento que queremos estilizar. Para isso usamos seletores, que veremos melhor no próximo tópico.

                
                    <head>
                        <style>
                            p {
                                color: red;
                            }
                        </style>
                    </head>

                    <body>
                        <p>Este texto será vermelho!</p>
                    </body>

Resultado:

Este texto será vermelho!

3 - Externo

A terceira maneira de se usar CSS é chamada de Externa. Pois é criado um outro documento separado do documento HTML apenas para os estilos, esse documento tem a extensão .css e deve ser linkado ao arquivo HTML utilizando-se a tag <link> dentro da tag <head> vejamos o exemplo:

No arquivo style.css:

                
                    p {
                        color: red;
                    }

No arquivo index.html:

                
                    <head>
                        <link rel="stylesheet" href="/src/css/style.css">
                    </head>

                    <body>
                        <p>Este texto será vermelho!</p>
                    </body>

Resultado:

Este texto será vermelho!

Repare que a tag <link rel="stylesheet" href="/src/css/style.css"> possui dois atributos: rel que determina o tipo de arquivo, nesse caso stylesheet ou folha de estilos; e href que determina o endereço do arquivo, neste caso o arquivo style.css está dentro de uma pasta chamada css que está dentro de outra pasta chamada src.

Seletores

Como vimos no tópico anterior, precisamos de uma maneira de identificar o elemento HTML que queremos alterar o estilo. Para isso usamos os chamados seletores. Existem muitos tipos de seletores e diversas regras e combinações para usá-los. Hoje veremos apenas alguns tipos mais simples, em uma próxima aula veremos outros mais complexos, como usar vários seletores em conjunto e algumas regras para determinar quais seletores tem prioridade.

1 - Tag

O seletor de Tag seleciona todos elementos declarados com aquela tag. Para usá-lo basta escrever o nome da tag e colocar as propriedas CSS entre colchetes:

No arquivo style.css:

                
                    p {
                        color: red;
                    }

No arquivo index.html:

                
                    <head>
                        <link rel="stylesheet" href="/src/css/style.css">
                    </head>

                    <body>
                        <p>Este texto será vermelho!</p>
                        <p>Este texto também será vermelho!</p>
                    </body>

Resultado:

Este texto será vermelho!

Este texto também será vermelho!

2 - Id

O seletor por Id utiliza o atributo id em uma tag HTML para selecionar aquele elemento. O atributo id deve ser único, ou seja, se existe um elemento com id="meuID", não pode existir outro elemento com o mesmo id.

Para selecionar um elemento por id, utilizamos a hashtag (#) seguida do id, veja o exemplo:

No arquivo style.css:

                
                    #paragrafo1 {
                        color: red;
                    }

No arquivo index.html:

                
                    <head>
                        <link rel="stylesheet" href="/src/css/style.css">
                    </head>

                    <body>
                        <p id="paragrafo1">Este texto será vermelho!</p>
                        <p>Este texto não será vermelho!</p>
                    </body>

Resultado:

Este texto será vermelho!

Este texto não será vermelho!

3 - Class

O seletor por Classe utiliza o atributo class em uma tag HTML para selecionar aquele elemento. O atributo class não precisa deve ser único, ou seja, podem existir vários elemento com class="minhaClasse".

Para selecionar um elemento por class, utilizamos o ponto (.) seguido do nome da classe, veja o exemplo:

No arquivo style.css:

                
                    .texto-vermelho {
                        color: red;
                    }

No arquivo index.html:

                
                    <head>
                        <link rel="stylesheet" href="/src/css/style.css">
                    </head>

                    <body>
                        <p class="texto-vermelho">Este texto será vermelho!</p>
                        <p class="texto-vermelho">Este texto também será vermelho!</p>
                        <p>Este texto não será vermelho!</p>
                    </body>

Resultado:

Este texto será vermelho!

Este texto também será vermelho!

Este texto não será vermelho!

4 - Universal

O seletor universal, como o próprio nome sugere, seleciona todos elementos HTML da página, independente de tag, clase ou id, sem exceção.

Para usar o seletor universal, utilizamos o asterístico (*), veja o exemplo:

No arquivo style.css:

                
                    * {
                        color: red;
                    }

No arquivo index.html:

                
                    <head>
                        <link rel="stylesheet" href="/src/css/style.css">
                    </head>

                    <body>
                        <p>Este texto será vermelho!</p>
                        <h6>Este texto também será vermelho!</h6>
                    </body>

Resultado:

Este texto será vermelho!

Este texto também será vermelho!

Agora que já aprendemos a usar alguns seletores, vamos ver mais algumas propriedades de estilo que podemos mudar.

Cor do Texto

Como já vimos podemos mudar a cor do texto utilizando a propriedade color. Veja o exemplo abaixo:

No arquivo style.css:

                
                    p {
                        color: red;
                    }

                    h6 {
                        color: blue;
                    }

No arquivo index.html:

                
                    <head>
                        <link rel="stylesheet" href="/src/css/style.css">
                    </head>

                    <body>
                        <p>Este texto será vermelho!</p>
                        <h6>Este texto será azul!</h6>
                    </body>

Resultado:

Este texto será vermelho!

Este texto será azul!

Cor de Fundo

Semelhante a cor do texto, podemos mudar também a cor do fundo, utilizando a propriedade background-color. Veja o exemplo abaixo:

No arquivo style.css:

                
                    p {
                        background-color: red;
                    }

                    h6 {
                        background-color: blue;
                    }

No arquivo index.html:

                
                    <head>
                        <link rel="stylesheet" href="/src/css/style.css">
                    </head>

                    <body>
                        <p>Este fundo será vermelho!</p>
                        <h6>Este fundo será azul!</h6>
                    </body>

Resultado:

Este fundo será vermelho!

Este fundo será azul!

Fontes

Existem algumas propriedades que mudam o visual da fonte. Veja a tabela abaixo:

Propriedade Utilidade
font-family Muda o tipo da fonte
font-style Deixa a fonte itálica
font-weight Deixa a fonte em negrito ou mais fina
font-size Muda o tamanho da letra
Veja o exemplo:

No arquivo style.css:

                
                    .arial {
                        font-family: Arial;
                    }

                    .italico {
                        font-style: italic;
                    }
                    .negrito {
                        font-weight: bold;
                    }

                    .tamanho32 {
                        font-size: 32px;
                    }

No arquivo index.html:

                
                    <head>
                        <link rel="stylesheet" href="/src/css/style.css">
                    </head>

                    <body>
                        <p class="arial">A fonte é Arial</p>
                        <p class="italico">A fonte está em itálico</p>
                        <p class="negrito">A fonte está em negrito</p>
                        <p class="tamanho32">A fonte está em tamanho 32px </p>
                    </body>

Resultado:

A fonte é Arial

A fonte está em itálico

A fonte está em negrito

A fonte está em tamanho 32px

text-align

Uma propriedade muito utilizada é text-align que determina o alinhamento do texto. Pode ser left (esquerda), right (direita), justify (justificado) ou center (centralizado). Veja o exemplo:

No arquivo style.css:

                
                    .esquerda {
                        text-align: left;
                    }
                    .direita {
                        text-align: right;
                    }
                    .justificado {
                        text-align: justify;
                    }
                    .centralizado {
                        text-align: center;
                    }

No arquivo index.html:

                
                    <head>
                        <link rel="stylesheet" href="/src/css/style.css">
                    </head>

                    <body>
                        <p class="esquerda">Texto alinhado a esquerda</p>
                        <p class="direita">Texto alinhado a direita</p>
                        <p class="justificado">Texto justificado</p>
                        <p class="centralizado">Texto centralizado</p>
                    </body>

Resultado:

Texto alinhado a esquerda

Texto alinhado a direita

Texto justificado

Texto centralizado

É mais fácil de visualizar a diferença entre os alinhamentos em textos mais longos (mais de duas linha) tente escrever um texto maior e ver a diferença entre os tipos de alinhamento.

text-decoration

Uma propriedade muito usada em links é text-decoration: none; para remover o sublinhado dos links. Veja o exemplo:

No arquivo style.css:

                
                    a {
                        text-decoration: none;
                    }

No arquivo index.html:

                
                    <head>
                        <link rel="stylesheet" href="/src/css/style.css">
                    </head>

                    <body>
                        <a href="https://www.google.com" target="_blank">Ir para o Google</a>
                    </body>

Resultado:

Ir para o Google

list-style-type

Uma propriedade muito usada é list-style-type: none; para remover os pontos na frente de cada item de uma lista. Veja o exemplo:

No arquivo style.css:

                
                    ul {
                        list-style-type: none;
                    }

No arquivo index.html:

                
                    <head>
                        <link rel="stylesheet" href="/src/css/style.css">
                    </head>

                    <body>
                        <ul>
                            <li>Item 1</li>
                            <li>Item 2</li>
                            <li>Item 3</li>
                        </ul>
                    </body>

Resultado:

  • Item 1
  • Item 2
  • Item 3

Tabelas

Em tabelas é comum utilizar a propriedade border-collapse: collapse; para que as bordas se juntem e não tenham espaço entre si. Para fazer as bordas é preciso usar a propriedade border: 1px solid black; nas céluas da tabela td. Veja o exemplo:

No arquivo style.css:

                
                    td {
                        border: 1px solid black;
                    }
                    table {
                        border-collapse: collapse;
                    }

No arquivo index.html:

                
                    <head>
                        <link rel="stylesheet" href="/src/css/style.css">
                    </head>

                    <body>
                        <table>
                            <tr>
                                <td>Célula 1</td>
                                <td>Célula 2</td>
                                <td>Célula 3</td>
                            <tr>
                            <tr>
                                <td>Célula 4</td>
                                <td>Célula 5</td>
                                <td>Célula 6</td>
                            <tr>
                        </table>
                    </body>

Resultado:

Célula 1 Célula 2 Célula 3
Célula 4 Célula 5 Célula 6
