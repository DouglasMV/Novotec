Aula 6 - Visual Studio Code

O que é o VS Code?

Visual Studio Code é um editor de código leve porém poderoso. Ele vem com diversas ferramentas que facilitam a escrita de código em várias linguagens. Além disso conta com uma enorme biblioteca de extensões que adicionam mais funcionalidades e suporte a mais linguagens de programação.

Abrindo uma pasta

Após iniciar o programa, no canto esquerdo superior, clique em File e depois em Open a Folder. Então escolha a pasta que deseja abrir para trabalhar. Isso é importante pois trabalhando na pasta onde estão os arquivos que queremos editar facilita encontrar e abrir esses arquivos.

Após isso uma barra lateral aparecerá com todos arquivos dentro da pasta que você acabou de abrir. Se você posicionar o cursosr do mouse sobre o nome da pasta na barra lateral, aperecem alguns icones. O primeiro é um atalho para criar um novo arquivo dentro da pasta, e o segundo para criar uma sub-pasta.

Emmet

Emmet é uma ferramenta que já vem instalada no VS Code. Ele serve para facilitar a escrita de códigos repetitivos. Vejamos alguns exemplos.

Primeiro crie um arquivo chamado index.html e um arquivo chamado style.css na sua pasta.

Agora no arquivo index.html digite ! e aperte a tecla Tab.

Isso gera um esqueleto padrão para um documento html. A maioria das tags você já conhece de aulas passadas, pesquise para que servem as tags que você não conhece.

Agora dentro da tag body digite o seguinte: ul>li*5 e aperte Tab. Isso cria uma tag ul com 5 tags li dentro. Bem mais rápido que digitar certo?

Agora digite table>(tr>th*4)+(tr>td*4)*10 e aperte Tab. Isso cria uma tabela com 11 linhas e 4 colunas. Impossível fazer isso mais rápido digitando linha por linha.

Agora digite ul>li{Item $}*10 e aperte Tab. Isso criar uma lista de 10 items, repare que o que estava entre {} aparece como o conteúdo de cada tag li, e no lugar do $ temos números de 1 a 10.

Agora digite p.azul e aperte Tab. Isso cria uma tag p com a classe "azul". Semelhnatemente se digitarmos h3#titulo e Tab, criamos uma tag h3 com o id "titulo".

Além de todos esses exemplos, se digitarmos apenas o nome da tag, por exemplo h1 e apertarmos Tab uma tag h1 é automaticamente criada, com a tag de abertura e de fechamento! Ainda repare que quando estamos digitando aparecem sugestões, podemos selecionar dentre elas usando as setas para cima e para baixo do teclado e apertar Tab para auto-completar. Repare que isso também funciona para propriedades e valores em CSS.

Emmet tem muitas funcionalidades para diversas linguagens. É impossível listar todas aqui. A melhor maneira de aprender é experimentando e praticando. Após algum tempo de prática você vai saber alguns atalhos de cabeça e isso economizará tempo na escrita de código.

Atalhos

Existem vários atalhos úteis em VS Code que podem facilitar tarefas e otimizar o seu tempo de trabalho. Veremos algumas aqui, para uma lista mais completa acesse essa lista.

Atalho Funcionalidade
Home Vai para o início da linha
End Vai para o final da linha
Crtl → ou ← Avança o cursor uma palavra
Shift → ou ← Seleciona uma letra
Ctrl Shift → ou ← Seleciona uma palavra
Ctrl L Seleciona uma linha inteira
Alt ↑ ou ↓ Mover a linha para cima ou para baixo
Ctrl ] ou [ Alinha o código para direita ou para esquerda
Ctrl F2 Seleciona todas ocorrências iguaisao que está selecionado
Alt Click Cria multiplos cursores
Ctrl / Comenta ou descomenta uma linha
Ctrl , Abre as opções de configuração
Ctrl B Abre ou Fecha a barra lateral
Ctrl Shift ´ Abre ou Fecha o terminal de comandos
