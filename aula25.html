Aula 25 - JS: Introdução

JavaScript é uma linguagem de programação que foi criada para fazer as páginas web ficarem mais "vivas".

Os programs escritos em JavaScript são chamados scripts. Eles são arquivos de texto que podem ser executados diretamente nos documentos HTML.

Com JavaScript é possível fazer muitas coisas nas páginas web, como por exemplo:

  • Adicionar ou remover HTML a página.
  • Modificar conteúdo existente.
  • Modificar estilos CSS
  • Reagir a ações do usuário (cliques, teclas, posição do mouse).
  • Enviar requsições a servidores pela rede.
  • Manipular dados no lado do cliente.

Onde escrever JavaScript?

Semelhante a CSS podemos escrever JavaScript de três maneiras: inline, interno e externo.

Por enquanto apenas veremos como escrever JavaScript interno e externo. Veremos inline quando aprendermos sobre eventos em outra aula.

Interno

Para escrever JavaScript internamento em um documento HTML usamos a tag <script></script> e escrevemos nosso código dentro dela. Geralmente escrevemos essa tag no final do documento HTML logo antes da tag </body> veja o exemplo:

No documento index.html:

                
                    <body>
                        <script>
                            alert("Olá Mundo!");
                        </script>
                    </body>

Externo

JavaScript externo consiste em criar um arquivo, por exemplo app.js, e conectá-lo ao documento HTML usando a tag script com o atributo src, veja o exemplo:

No documento index.html:

                
                    <body>
                        <script src="app.js"></script>
                    </body>

No documento app.js:

                
                    alert("Olá Mundo!");

Comentários

Em JavaScript escrevemos comentários entre /* Comentário aqui */. Tudo que estiver dentro do comentário é ignorado pelo navegador na hora de executar o código. Também podemos escrever um comentário de uma linha usando duas barras no começo da linha, veja o exemplo:

No documento index.html:

                
                    <body>
                        <script src="app.js"></script>
                    </body>

No documento app.js:

                
                    // Comentário de uma linha
                    
                    alert("Olá Mundo!");
                    
                    /* 
                    Comentário de várias linhas!
                    Tudo que está aqui será ignorado!
                    alert("Não vou aparecer!");
                    */

Ponto e Vírgula é Opcional

Repare nos exemplos anteriores que usamos ponto e vírgula no final de uma declaração. Em JavaScript não é obrigatório utilizar ponto e vírgula no final de cada declaração, é opcional, depende do gosto do programador. Existem alguns casos específicos que não usar ponto e vírgula pode causar problemas, para evitar isso nós utilizaremos.

Console

É comum ocorrerem erros em um script. Geralmente esses erros não são visualizados pelo usuário. Pois o local onde os navegadores mostram esse erro é no console. No Google Chrome para abrir o console aperte a tecla F12.

Podemos também digitar código JavaScript diretamente no console, se queremos testar algo rapidamente por exemplo.

console.log()

Podemos escrever uma mensagem no console utilizando console.log em nosso script. Veja o exemplo:

No documento app.js:

                
                   console.log("Olá Mundo!");

Variáveis

Variáveis são usadas para guardar uma informação.

Existem 4 coisas importantes para se saber sobre variáveis:

1 - Nome:

Toda variável tem um nome, que serve para identificá-la, para que ela seja usada em várias partes do script.

O programador (você) escolhe o nome da variável, porém deve obdecer três regras:

I - O nome pode conter apenas letras (sem acento), números, $ e _

II - O nome não pode começar com um número

III - O nome não pode ser uma palavra reservada

Geralmente usamos o padrão camelCase para nomes de variáveis com mais de uma palavra. Isso é, a primeira palavra é minúscula e as outras começam com letra maiúscula. Por exemplo: minhaVariavel, minhaVariavelMuitoLegal. Mas tome cuidado, nomes muito grandes podem dificultar o entendimento do seu script.

2 - Declaração:

Declarar uma variável é criá-la, reservar um nome (e um espaço na memória do computador) para ela.

Para declarar uma variável usamos a palavra chave let seguido do nome da variável. Veja o exemplo, onde criamos uma variável chamada sobrenome:

No documento app.js:

                
                    let sobrenome;

Também podemos criar variáveis com as palavras chave var ou const. Existem algumas diferenças porém vamos usar apenas let que servirá para nós em todos casos. (Pesquise sobre as diferenças entre const, var e let, aprender é divertido!)

3 - Atribuição:

Variáveis guardam informações ou valores, para isso precisamos atribuir um valor para ela. Para atribuir um valor escrevemos o nome da variável, seguido de um sinal de igual (=) segudio do valor a ser guardado na variável.

Podemos atribuir um valor a variável na declaração ou em outra parte do script. Veja o exemplo:

No documento app.js:

                
                    let sobrenome = "Mariano Valero";
                    let idade;
                    idade = 28;

4 - Acesso:

Para usar uma variável em outras partes do script, precisamos acessá-la. Fazemos isso usando o nome da variável. Adicione a linha abaixo no exemplo anterior.

No documento app.js:

                
                    alert(sobrenome);

Repare na diferença entre este exemplo e o primeiro exemplo desta aula. Neste usamos apenas o nome da variável, no outro usamos aspas e escrevemos uma frase dentro.

Variáveis podem ser de vários tipos diferentes, por exemplo, podem ser uma cadeia de caracteres (como o sobrenome no exemplo), ou um número (como idade no exemplo), dentre outros. Veremos a seguir quais são os tipos em JavaScript.

Tipos

number

O tipo number representa um número, pode ser inteiro ou decimal. Mais adiante vamos aprender operações aritméticas, que só podem ser realizadas com números. Veja alguns exemplos de variáveis do tipo number:

No documento app.js:

                
                    let idade = 22;
                    let valor = 10.25;
                    let altura = 1.75;

string

Uma string é uma cadeia de caracteres. Para declarar um string é preciso usar aspas. Podemos usar três tipos de aspas: aspas simples, aspas duplas e acento grave. Veja o exemplo:

No documento app.js:

                
                    let nome = 'Douglas';
                    let sobrenome = "Mariano Valero";
                    let filme = `Debi e Lóide`;

boolean

O tipo boolean pode receber apenas dois valores: false ou true. Usamos isso para definir se algo é verdadeiro ou falso, isso será importante em uma próxima aula onde aprenderemos condicionais. Veja o exemplo:

No documento app.js:

                
                    let online = true;
                    let premium = false;

null

O tipo null é um tipo especial que contém apenas o valor null. Representa nada ou vazio. Geralmente utilizamos quando queremos remover um valor de uma variável, veja o exemplo:

No documento app.js:

                
                    let supervisor = João;
                    supervisor = null;

undefined

O tipo undefined é um tipo especial que contém apenas o valor undefined. Representa não definido. Apesar de ser possível, geralmente não o utilizamos para atribuir um valor para uma variável (utilizamos null). Pois undefined é o valor padrão para variáveis que foram declaradas mas não tiveram nenhum valor atribuído, veja o exemplo:

No documento app.js:

                
                    let minha_variavel;
                    alert(minha_variavel);

object

O tipo object representa um objeto. Um objeto em JavaScript pode conter várias propriedades e métodos (estudaremos isso em uma próxima aula). Para declarar um objeto usamos chaves {} e dentro das chaves declaramos proprieades usando pares nome/valor separados por dois pontos. Para acessar uma propriedade usamos o nome do objeto, seguido de um ponto, e seguido do nome da propriedade. Veja o exemplo:

No documento app.js:

                
                    let usuario = {
                        nome: "Douglas",
                        sobrenome: "Mariano Valero",
                        idade: 28;
                    }

                    alert(usuario.nome);

array

Tecnicamente array não é um tipo em JavaScript. Na realidade é um tipo especial de object. Mas vamos tratá-lo aqui pois é um tipo especial muito utilizado.

Array é um conjunto de valores ordenados. Para declarar um array escrevemos os valores entre colchetes, separados por vírgula, e para acessar esses valores usamos o nome do array e a posição do valor entre colchetes. Porém é preciso lembrar de algo muito importante: arrays começam do 0. Além disso todo array possui uma propriedade especial, chamada length, que retorna o número de elementos de um array. Veja o exemplo:

No documento app.js:

                
                    let cores = ["azul", "amarelo", "vermelho"];
                    alert(cores[0]);
                    alert(cores[1]);
                    alert(cores[2]);
                    alert(cores[3]); // undefined
                    alert(cores.length);

typeof

Podemos usar typeof e um valor ou uma variável para saber seu tipo. Tenha em mente que o typeof nem sempre funciona, existem alguns casos muito estranhos. Mas na maior parte das vezes ele é confiável. Veja os exemplos:

No documento app.js:

                
                    let nome = "Douglas"
                    alert(typeof nome); //string
                    alert(typeof "Olá"); //string
                    alert(typeof "3"); //string
                    alert(typeof 3); //number

                    let usuario ={
                        nome: "Doglas",
                        idade: 28
                    }
                    alert(typeof usuario); //object
                    alert(typeof usuario.nome); //string
                    alert(typeof usuario.idade); //number

                    alert(typeof [2, 2, 4, 5,]); //object, pois array é um tipo especial de object
                    alert(typeof undefined); // undefined
                    alert(typeof null); // object????? É um daqueles casos estranhos, mas raramente vamos encontrar

Operadores

Vamos falar agora de operadores. São caracteres (+, -, *, /, =, etc..) que realizam uma operação em variáveis ou valores.

Atribuição

Já conhecemos o operador de atribuição. Ele serve para guardarmos um valor em uma variável. Para usá-lo precisamos de uma variável a esquerda dele e um valor de qualquer tipo (number, string, array, object, boolean, etc...) a direita. Veja o exemplo:

No documento app.js:

                
                    let nome = "Douglas";
                    let online = false;
                    online = true;

Concatenação de strings

Concatenar é "juntar". Podemos juntar strings, por exemplo, algo que queremos escrever na tela com uma variável. Para isso usamos o sinal de +. Veja o exemplo:

No documento app.js:

                
                    let nome = "Douglas";
                    alert("Olá " + nome + ", bem-vindo!");

Soma

O sinal de + também é usado para somar números. Tome cuidado aqui, pois se um dos dois operandos for uma string, o que acontecerá é uma concatenação, veja os exemplos:

No documento app.js:

                
                    let idade = 28;
                    alert(idade + 1);  //  29
                    alert(2+2); //  4
                    alert(1+"1"); // 11, pois o segundo "1" é uma string

Subtração

O sinal de - é usado para subtrair números. Veja os exemplos:

No documento app.js:

                
                    let idade = 28;
                    alert(idade - 1);  //  27
                    alert(2-2); //  0
                    alert(1-"1"); // 0, por quê?????

Repare no terceiro alert do exemplo, apesar do segundo valor ser uma string a subtração ocorreu corretamente. Isso acontece por dois motivos:

1 - não existe nenhuma operação especial com o sinal - para strings.

2 - quando usamos esse operador ele automaticamente converte qualquer tipo que não seja um number para um number.

Apesar de funcionar, geralmente não é recomendado realizar subtração com valores que não sejam do tipo number. Nesse exemplo o resultado foi o esperado porque a conversão automática resultou no que é esperado, mas nem sempre é esse o caso. Por isso evite fazer operações aritméticas (soma, subtração, divisão, multiplicação, etc..) com strings.

Multiplicação

O sinal de * é usado para multiplicar números. Veja os exemplos:

No documento app.js:

                
                    let idade = 28;
                    alert(idade * 0.5);  //  14
                    alert(2*2); //  4
                    alert(1*"1"); // 1, conversão automática, semelhante a subtração

Divisão

O sinal de / é usado para dividir números. Veja os exemplos:

No documento app.js:

                
                    let idade = 28;
                    alert(idade / 2);  //  14
                    alert(2/2); //  1
                    alert(1/"1"); // 1, conversão automática, semelhante a subtração

Exponenciação

O sinal de ** é usado para exponenciar números. Veja os exemplos:

No documento app.js:

                
                    alert(10**2); //  100
                    alert(2**3); //  8

Resto da divisão

Um operador que provavelmente você não está acostumado é o % ou resto da divisão. O resultado dele é o resto da divisão inteira do número a esquerda pelo número da direita. Por exemplo 32%3 é 2, pois 32 dividido por 3 dá 10 e sobram (resto) 2.

Pode não parecer mas esse operador é muito utilizado em programação.

No documento app.js:

                
                    let par = 22;
                    let impar = 33;
                    alert(par%2);   // 0
                    alert(impar%2); // 1

Incremento/Decremento

É muito comum em programação precisarmos adicionar 1 ou subtrair 1 de uma variável. Por isso existem os operadores de incremento ++ e de decremento --. Veja o exemplo:

No documento app.js:

                
                    let visitantes = 0;
                    visitantes++;
                    alert(visitantes); // 1
                    alert(visitantes++) // 1
                    alert(++visitantes) // 3

Repare que podemos usar esses operadores antes ou depois do nome da variável. A diferença é que se usarmos antes ele primeiro mostra o valor atual e depois adiciona/subtrai 1, se usarmos o operador antes do nome da variável ele primeiro adiciona/subtrai 1 e depois mostra o valor.

Troque os operadores para -- e veja o que acontece.

Modificar diretamente

É muito comum precisarmos modificar uma variável e salvar o novo valor na mesma variável. Podemos fazer isso assim:

No documento app.js:

                
                    let total = 0;   
                    total = total + 19.99;
                    alert(total);

E perfeitamente comum achar esse tipo de código confuso a primeira vez, pois estamos utilizando a variável "total" duas vezes na mesma linha. Porém se você para um pouco para pensar, vai perceber que ela é usada de duas maneiras diferentes: a direita do = usamos o valor salvo nela (0), somamos 19.99 a ele e salvamos o resultado na própria váriavel, a esquerda do =. Percebeu como ela é usada para acessar o valor, e depois para atribuir um novo valor? E sim a ordem parece estranha pois é da direita pra esquerda, mas isso é porque o operador de adição (+) tem preferência sobre o operador de atribuição (=).

A forma de modificar uma variável diretamente é a seguinte:

                
                    let total = 0;   
                    total += 19.99;
                    alert(total);

Usamos o operador da soma e de atribuição juntos, a ordem é importante, o operador de atribuição sempre vem depois. Também podemos fazer isso com outros operadores aritméticos (-, /, *, %, **), experimente.

Números que não são números mas são números???

O título dessa sessão é meio confuso mas tudo fará sentido quando você acabar de lê-la. Em JavaScript existem alguns valores especiais quando trabalhamos com números. Vamos vê-los agora.

Infinity

Um desses valores especiais é Infinity, que significa um número maior que qualquer outro número.. Você talvez já tenha ouvido um professor de Matemática dizer que é proibido ou impossível dividir por zero, por exemplo: 1/0. Mas em JavaScript isso é possível, veja o exemplo:

No documento app.js:

                
                    let num = 1/0;
                    alert(num);
                    alert(typeof num);

Repare que o valor da divisão foi o valor especial Infinity. E repare que o typeof de Infinity é number. Troque 1 por -1 no exemplo anterior. Repare que o resultado agora é -Infinity, que significa um número menor que qualquer outro número.

NaN

Outro valor numérico especial é NaN, que siginifica "Not a Number", em português "Não um Número". Esse valor é obtido como resultado de operações aritméticas (+, -, *, /, **, %) que envolvam valores não numéricos. Veja os exemplos:

No documento app.js:

                
                    let num = 5 * "laranja";
                    alert(num);
                    alert(typeof num);

Vimos que quando um dos operandos não é do tipo number, ele é automaticamente convertido para um número. Porém quando temos uma string do tipo "10" é possível converter, já no exemplo temos a string "laranja" e o JavaScript não sabe converter isso para um número, obviamente. Então o resultado da operação é NaN, "Não um Número".

Até ai tudo faz sentido. Porém quando usamos typeof em NaN temos o resultado "number", ou seja, NaN é do tipo number. Ok, admito, isso é bem confuso. Mas basta você pensar que NaN é um valor especial que representa um resultado de uma operação com números que não foi possível calcular, muito provavelmente porque um dos valores não era realmente um número ou algo que pudesse ser convertido para um número.

Nas próximas aulas veremos casos em que NaN será importante. Por exemplo, imagine que pedimos para o usuário digitar sua idade, e ele digita "laranja", e depois precisamos fazer uma operação aritimética com esse valor, precisamos verificar se a idade é um número, ou seja se não é um NaN.

Conversões de Tipos

Vimos que algumas vezes os valores são convertidos automaticamente, por exemplo quando escrevemos 10 - "5" o valor "5" é uma string, mas é convertido automaticamente para o tipo number, pois é o único tipo aceitável em uma operação de subtração.

Mas as vezes queremos que um valor seja convertido e isso não acontece automaticamente, podendo gerar resultados indesejáveis, veja o exemplo:

No documento app.js:

                
                    let soma = 1 + "1";
                    alert(soma);

Nesse exemplo o resultado é a string "11", pois o programa entendeu isso como uma concatenação de strings, e na verdade convertou o number 1 para string e juntou com a outra string "1" resultando na string "11". Mas claramente não era isso que queriamos fazer. O resultado que esperavamos era 2. Para isso precisamos converter a string "1" para o tipo number. A seguir veremos como fazer isso.

Number()

Para converter um valor para o tipo number usamos Number(minhaString) e dentro dos parenteses colocamos o valor ou a variável que queremos converter.

No documento app.js:

                
                    let soma = 1 + Number("1");
                    alert(soma);

Teoricamente podemos converter qualquer tipo de valor para number, mas geralmente usamos isso para string.

String()

Podemos também converter um valor para string, utilizando String(meuValor). Veja o exemplo:

No documento app.js:

                
                    let num = 10;
                    num = String(num);
                    alert(typeof num);

Boolean()

Também é possível converter um valor para o tipo boolean. Lembre-se que boolean tem apenas dois valores possíveis: true e false. É raro usarmos Boolean() para conversões. As conversões para o tipo boolean geralmente acontecem automaticamente em condicionais e loops, que aprenderemos na próxima aula. Porém é importante conhecer alguns valores que aparecem muito e qual é a conversão dels para boolean.

Nesse primeiro exemplo vemos vários valores que convertem para o boolean true. Observe que alguns não são tão obvios:

                
                    alert(Boolean(1));
                    alert(Boolean("1"));
                    alert(Boolean("Uma string não vazia"));
                    alert(Boolean(" "));
                    alert(Boolean("0"));
                    alert(Boolean("false"));
                    alert(Boolean("null"));
                    alert(Boolean("undefined"));
                    alert(Boolean([a, b, c, d]));
                    alert(Boolean({chave: "valor"}));
                    alert(Boolean([]));
                    alert(Boolean({}));

Nesse segundo exemplo temos valores que convertem para o boolean false:

                
                    alert(Boolean(0));
                    alert(Boolean(-0));
                    alert(Boolean(''));
                    alert(Boolean(""));
                    alert(Boolean(``));
                    alert(Boolean(null));
                    alert(Boolean(undefined));
                    alert(Boolean(NaN));
                    alert(Boolean(false));

                    let minhaVariavel;
                    alert(Boolean(minhaVariavel));

O objeto global window

Todo ambiente JavaScript possui um objeto global, um objeto que possui propriedades e funções que podem ser acessadas de qualquer parte do código.

Nos navegadores esse objeto global é o objeto window. Esse objeto possui algumas propriedades e funções padrões que podemos usar.

alert()

Não vimos ainda o que é uma função (ou método), veremos em uma aula posterior. Porém mesmo sem saber já utilizamos várias vezes um dos métodos do objeto global window: alert(). O que esse método faz é abrir uma caixa de diálogo no navegador e mostrar o texto que usamos entre parenteses. Veja o exemplo:

No documento app.js:

                
                    alert("Olá mundo!");

prompt()

O método prompt() é usado para receber um valor do usuário e salvá-lo em uma variável, entre parenteses digitamos uma string que será exibida para o usuário. Veja o exemplo:

No documento app.js:

                
                    let nome;
                    nome = prompt("Digite seu nome:");

                    alert("Olá, " + nome);

confirm()

O método confirm abre uma caixa de diálogo com dois botões "OK" e "Cancel", e salva um boolean em uma variável. Se o usuário apertar "OK" salva o valor true, se o usuário apertar "Cancel" salva false. Veja o exemplo:

No documento app.js:

                
                    let deMaior;
                    deMaior = confirm("Têm mais de 18 anos?");

                    alert("Maior de 18: " + deMaior);

open() e close()

O método open() é usado para abrir outra janela. Atribuimos ele a uma variável para depois poder fechar a janela. Veja o exemplo:

Digite no console do navegador:

                
                    let google;

                    google = window.open("http://www.google.com/");

Agora digite:

                
                    google.close();

Talvez isso não funcione em alguns navegadores, principalmente porque a maioria bloqueia esses métodos para evitar que sites abusem deles para abrir novas janelas de propagandas ou sites maliciosos (que contém vírus, spyware ou outros malwares).

O objeto document

Outro objeto importante em navegadores é o objeto document. É através dele que conseguimos acessar os elementos HTML, assim podemos modificar, adicionar ou remover elementos HTML.

Não veremos como fazer isso agora, teremos uma outra aula só para isso. Por enquanto veremos apenas um método:

document.write

Esse método nos permite escrever código HTML direto na nossa página, veja o exemplo:

No documento app.js:

                
                    document.write("<h1>Um cabeçalho h1!</h1>")

Porém esse método é um pouco limitado. Perceba que quando usamos ele todo conteúdo HTML que estava na página é excluido e apenas é exibido o novo conteúdo passado dentro dos parenteses. Isso não é ideal, por isso aprenderemos outros métodos em uma das nossas próximas aulas.

Esses métodos (document.write, alert, prompt, confirm, open, close) serão úteis para nós agora no começo enquanto estamos aprendendo o básico de programação com JavaScript. Porém mais para frente aprenderemos maneira muito melhores de interagir com o usuário e com a página HTML em si. Na prática esses métodos não são utilizados (exceto em sites muito antigos), utilizaremos apenas para fins de aprendizagem mesmo.

Exercícios:

1 - Olá ?????

Faça um script que pergunta o nome do usuário e depois mostra "Olá Nome", mas no lugar de "Nome" mostra o nome digitado.

2 - Quando eu nasci?

Escreva um script que pergunta a um usuário quantos anos ele faz esse ano (em 2019) e depois o programa mostra para o usuário que ano ele nasceu.

3 - Escolha a cor de fundo!

Escreva um script que pede ao usuário para digitar valores de 0 a 255 para as cores red, green e blue, e depois mude a cor de fundo do body para esses valores.

4 - Filmes favoritos!

Escreva um script que pergunta o nome de 5 filmes para o usuário e depois mostre esses 5 filmes em uma lista.
