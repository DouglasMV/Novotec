Aula 10 - Layout

Layout é a forma com que os elementos de uma página web são distribuídos. O posicionamento dos elementos. Nesta aula aprenderemos algumas propriedades de CSS que nos permitem controlar como cada elemento da página é posicionado.

Fluxo

Um navegador mostra os elementos de uma página HTML, que não possui propriedades CSS, da esquerda para direita, de cima para baixo, na ordem em que existem no documento HTML. Isso é chamado de fluxo dos elementos em HTML.

CSS possui propriedades que mudam o posicionamento, ou o fluxo, de elementos HTML. Nessa aula veremos apenas 5 dessas propriedades: position; display; z-index; flot; clear. Em aulas futuras veremos também FlexBox e Grid, que também são propriedades de posicionamento de elementos.

Display

Todo elemento HTML tem um modo de exibição (display) padrão. Alguns elementos ocupam a largura total de seu elemento pai, independente do tamanho de seu conteúdo e portanto não possuem nenhum elemento ao seu lado, por exemplo um cabeçalho <h1>. Outros elementos ocupam apenas o tamanho horizontal necessário para exibir seu conteúdo e podem ser colocados diretamente do lado de outros elementos, como por exemplo <em>.

Nessa aula estudaremos 4 valores possíveis para propriedade display: inline; block; inline-block; none. E conheceremos as tags <div> e <span>.

display: inline;

O valor inline na propriedade display faz com que o elemento ocupe apenas a largura necessária para exibir seu conteúdo (é impossível mudar a largura, a propriedade width não muda nada), e não exige que o elemento esteja em uma nova linha, ou seja podem haver outros elementos ao lado dele. Alguns exemplos de tags que tem display: inline como valor padrão são: <em>, <a>, <strong> e a tag <span> é usada como um elemento inline genérico, ou seja, que não tem uma função ou significado específico definido na navegador, ao contrário por exemplo da tag <a> que tem a função específica de conter um link. Veja o exemplo abaixo e repare como os elementos são exibidos por padrão:

                
                    <p>
                        Esse parágrafo têm um <a href="#">link</a>
                        e também uma palavra em <strong>destaque</strong>!
                    </p>

display: block;

Alguns elementos não são exibidos na mesma linha que outros elementos próximos a ele. Esse tipo de elemento ocupa toda linha por padrão, mas pode ser mudado com a propriedade width, que vimos na aula 8. Eles são chamods de elementos em bloco, pois tem a propriedade display: block como padrão. Alguns exemplos de tags que tem display: block como valor padrão são: <p>, <h1> a <h6>, <ul> e a tag <div> é usada como um elemento block genérico, ou seja, que não tem uma função ou significado específico definido na navegador. Veja o exemplo abaixo e repare como os elementos são exibidos por padrão:

                
                    <p>
                        Esse parágrafo é um bloco.
                    </p>
                    <div>
                        Essa div é um outro bloco.
                    </div>

Agora usando CSS tente mudar a propriedade display dos dois exemplos anteriores e repare o que acontece. Tente também mudar a propriedade width dos elementos que tem display: inline e repare que nada acontece.

display: inline-block;

O valor inline-block na propriedade display mistura características dos dois outros valores vistos (inline e block). Elementos inline-block podem ser exibidos na mesma linha que outros (característica de inline), mas pode ter sua largura (width) alterada(característica de block). O melhor exemplo de elemento inline-block é a tag <img>. Veja o exemplo abaixo:

                
                    <img src="https://i.imgur.com/CVarvWs.jpg" alt="Um gato">
                    <img src="https://i.imgur.com/CVarvWs.jpg" alt="Um gato">
                    <img src="https://i.imgur.com/CVarvWs.jpg" alt="Um gato">

Altere a propriedade width das imagens e veja o que acontece.

display: none;

Usar o valor none na propriedade display faz com que o elemento não seja exibido. Já vimos algo parecido com visibility: hidden; aula passada. A grande diferença aqui é que com dispaly: none; o elemento não ocupa o espaço que ele ocupava antes na tela, já com visibility: hidden; ele continua ocupando o espsço, mas fica invisível.

Tente mudar essas propriedades nos exemplos anteriores e repare na diferença entre as duas.

Position

A propriedade position determina a posição do elemento em relação ao fluxo. Ou seja, existe uma ordem em que os elementos são exibidos (o fluxo), caso queira que um elemento esteja em uma posição diferente da padrão, é preciso alterar a propriedade position. Veremos 5 valores para essa propriedade: static; relative; absolute; fixed; e sticky.

position: static;

É o valor padrão. Não muda nada. existe apenas ara casos de conflito de seletores CSS (veremos em outra aula).

position: relative;

Esse valor permite que o elemento seja posicionado relativamente a sua posição padrão (static). Isso é feito usando as propriedades: top; bottom; left; right. Essas propriedades movem o elemento para direção oposta ao seu nome. é mais fácil entender com um exemplo:

No arquivo style.css:

                
                    #div1 {
                        height: 300px;
                        width: 300px;
                        background-color: red;
                    }
                    #div2 {
                        height: 300px;
                        width: 300px;
                        background-color: blue;
                        position: relative;
                        top: 50px;
                        left: 100px;
                    }
                    #div3 {
                        height: 300px;
                        width: 300px;
                        background-color: green;
                    }

No arquivo index.html:

                
                    <div id="div1"></div>
                    <div id="div2"></div>
                    <div id="div3"></div>

Tente mudar os valores e usar várias combinações dessas propriedades. Repare que valores negativos podem ser usados. E repare que Top e Left tem prioridade sobre Bottom e Right respectivamente. Repare que a posição da #div3 não muda, continua respetando o espaço "original" ocupado pela #div2.

position: absolute;

Quando usar position: absolute; em um elemento ele será ignorado pelos outros elementos dentro do fluxo. E o elemento será posicionado em relação a posição do seu elemento pai (usando as propriedades top, left, bottom e right). Veja o exemplo abaixo:

No arquivo style.css:

                
                    #div1 {
                        height: 300px;
                        width: 300px;
                        background-color: red;
                    }
                    #div2 {
                        height: 300px;
                        width: 300px;
                        background-color: blue;
                        position: absolute;
                        top: 50px;
                        left: 100px;
                    }
                    #div3 {
                        height: 300px;
                        width: 300px;
                        background-color: green;
                    }

No arquivo index.html:

                
                    <div id="div1"></div>
                    <div id="div2"></div>
                    <div id="div3"></div>

Repare que a #div3 ignora o espaço "original" da #div2, e toma seu lugar no fluxo como se #div2 não existisse. Tente alterar os valores de top e left. Tente usar bottom e right também. No arquivo HTML, coloque a #div2 dentro da #div3 e repare que agora a posição da #div2 é em relação a #div3 que é sua tag pai.

position: fixed;

Quando usamos position: fixed; em um elemento, ele é posicionato em relação ao viewport (a parte da tela onde encontra-se o conteúdo visível do site). E ainda o elemento é fixado (por isso o nome fixed) naquela posição da tela, ou seja, mesmo que o usuário role a tela para baixo o elemento não sai do lugar. Veja o exemplo:

No arquivo style.css:

                
                    #div1 {
                        height: 100vh;
                        width: 100vw;
                        background-color: red;
                    }
                    #div2 {
                        height: 50px;
                        width: 100vw;
                        background-color: blue;
                        position: fixed;
                        top: 0;
                        left: 0;
                    }
                    #div3 {
                        height: 100vh;
                        width: 100vw;
                        background-color: green;
                    }

No arquivo index.html:

                
                    <div id="div1"></div>
                    <div id="div2"></div>
                    <div id="div3"></div>

Hmmm... Essa barra azul fixa no topo da página te lembra de algo?

position: sticky;

Quando usamos position: sticky; é como se o elemento altera-se entre position: relative; e position: fixed. Ou seja, ele começa posicionado em relação a sua posição "original", e quando o usuário rola o site para baixo o elemento se fixa naquela posição. Veja o exemplo para entender melhor:

No arquivo style.css:

                
                    #div1 {
                        height: 30vh;
                        width: 100vw;
                        background-color: red;
                    }
                    #div2 {
                        height: 50px;
                        width: 100vw;
                        background-color: blue;
                        position: sticky;
                        top: 0;
                        left: 0;
                    }
                    #div3 {
                        height: 100vh;
                        width: 100vw;
                        background-color: green;
                    }
                    #div4 {
                        height: 100vh;
                        width: 100vw;
                        background-color: purple;
                    }

No arquivo index.html:

                
                    <div id="div1"></div>
                    <div id="div2"></div>
                    <div id="div3"></div>
                    <div id="div4"></div>

Parece útil para um site com um banner ou cabeçalho principal e um menu abaixo desse banner que fica fixo conforme rolamos para baixo, não é mesmo?

Z-index

A propriedade z-index serve para deixar elementos na frente, ou atrás de outros elementos que o sobrepõem. Ela recebe como valor um número inteiro, quanto maior o número mais a frente o elemento será exibido. Um fato importante é que a propriedade z-index não funciona em elementos com a propriedade position: static; (o valor padrão), por isso é preciso mudar a position para outro valor (relative, absolute, fixed ou sticky). Veja o exemplo abaixo:

No arquivo style.css:

                
                    div {
                        height: 300px;
                        width: 300px;
                        position: absolute;
                    }
                    #div1 {
                        background-color: red;
                        top: 10px;
                        left: 10px;
                        z-index: -10;
                    }
                    #div2 {
                        background-color: blue;
                        top: 20px;
                        left: 20px;
                        z-index: 0;
                    }
                    #div3 {
                        background-color: green;
                        top: 30px;
                        left: 30px;
                        z-index: 6;
                    }
                    #div4 {
                        background-color: purple;
                        top: 40px;
                        left: 40px;
                        z-index: 9999;
                    }

No arquivo index.html:

                
                    <div id="div1"></div>
                    <div id="div2"></div>
                    <div id="div3"></div>
                    <div id="div4"></div>

Mude os valores de z-index, veja o que acontece.

Float

Até agora aprendemos como posicionar um elemento exatamente onde queremos. Mas se você quiser apenas mover um elemento o mais possível para a esquerda, ou o mais possível para direita, podemos usar a propriedade float. Além disso os elementos devem ter a propriedade width, e usando float os elementos podem ocupar a mesma linha, mesmo sendo display: block. Porém apenas é possível usar essa propriedade para elementos com a propriedade position com os valores static (padrão) ou relative. Veja o exemplo:

No arquivo style.css:

                
                    div {
                        height: 300px;
                        width: 300px;
                        position: static; /*Não necessário, só para ficar claro*/
                    }
                    #div1 {
                        background-color: red;
                        float: left;                        
                    }
                    #div2 {
                        background-color: blue;
                        float: right;                        
                    }
                    #div3 {
                        background-color: green;
                        float: left;                        
                    }
                    #div4 {
                        background-color: purple;
                        float: right;
                    }

No arquivo index.html:

                
                    <div id="div1"></div>
                    <div id="div2"></div>
                    <div id="div3"></div>
                    <div id="div4"></div>

Repare que os elementos com float: left; são posicionados da esquerda para a direita de acordo com a ordem deles no documento HTML. Já os elementos com float: right; são posicionados da direita para esquerda segundo a ordem que aparecem no documento HTML.

Se float parecer um pouco confuso no começo não se preocupe, essa é uma daquelas situações que temos uma boa e uma má notícia. A má notícia é que sim é um pouco confuso e nem sempre o resultado que obtemos é o que esperavamos na nossa cabeça. A boa notícia é que existem novas maneiras mais fáceis para fazer isso, usando Flex e Grid, que aprenderemos em aulas futuras.

Clear

A propriedade clear é usada juntamente com a propriedade float, quando existem vários elementos com a propriedade float, e queremos que um certo elemento não tenha outros elementos de um ou dos dois lados. Veja o exemplo, isso ficará bem mais claro.

No arquivo style.css:

                
                    div {
                        height: 300px;
                        width: 300px;
                        position: static; /*Não necessário, só para ficar claro*/
                    }
                    #div1 {
                        background-color: red;
                        float: left;                        
                    }
                    #div2 {
                        background-color: blue;
                        float: left;
                        clear: left;
                    }
                    #div3 {
                        background-color: green;
                        float: right;                        
                    }
                    #div4 {
                        background-color: purple;
                        float: right;
                        clear: right;
                    }

No arquivo index.html:

                
                    <div id="div1"></div>
                    <div id="div2"></div>
                    <div id="div3"></div>
                    <div id="div4"></div>
