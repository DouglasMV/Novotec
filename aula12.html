Aula 12 - Opacity

A propriedade opacity determina a opacidade/transparência de um elemento. Ela recebe um valor entre 0 e 1. Quanto maior o valor mais opaco (visível) é o elemento. Quanto menor o valor mais transparente (invisível) é o elemento. Veja o exemplo:

No arquivo style.css:

                
                    #div1 {
                        height: 300px;
                        width: 300px;
                        background-color: red;
                        z-index: 1;
                    }
                    #div2 {
                        height: 300px;
                        width: 300px;
                        background-color: blue;
                        z-index: 2;
                        opacity: 0.5;
                        position: absolute;
                        top: 50px;
                        left: 50px;
                    }

No arquivo index.html:

                
                    <div id="div1"></div>
                    <div id="div2">Olá Mundo!</div>

Repare que a div com fundo azul tem opacidade 0.5, assim podemos ver a div vermelha atras dela.

Podemos combinar opacity com o pseudo-seletor hover, que vimos aula passada, para criar um efeito legal. Veja o exemplo:

No arquivo style.css:

                
                    img {
                        width: 400px;
                        opacity: 0.3;
                    }
                    img:hover {
                        opacity: 1;
                    }

No arquivo index.html:

                
                    <img src="https://i.imgur.com/CVarvWs.jpg" alt="Um gato">

Passe o mouse em cima da imagem e veja o que acontece.

Opacidade em cores

Além de alterar a opacidade de todo elemento com a propriedade opacity, podemos alterar a opacidade apenas das cores, nos lugares onde usamos cores, por exemplo: color, background-color, border. Para isso precisamos usar cores no formato rgb mas com um quarto valor, esse quarto valor é a opacidade. Veja o exemplo:

No arquivo style.css:

                
                    #div1 {
                        height: 300px;
                        width: 300px;
                        background-color: red;
                        z-index: 1;
                    }
                    #div2 {
                        height: 300px;
                        width: 300px;
                        background-color: rgb(0, 0, 255, 0.5);
                        z-index: 2;
                        position: absolute;
                        top: 50px;
                        left: 50px;
                    }

No arquivo index.html:

                
                    <div id="div1"></div>
                    <div id="div2">Olá Mundo!</div>

Repare que o resultado é parecido com o primeiro exemplo. A diferença é que o conteúdo, o texto, não sofreu efeito de opacidade dessa vez, apenas a cor do fundo.
