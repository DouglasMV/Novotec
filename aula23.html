Aula 23 - @import

Modularização

Modularização é um conceito usado em desenvolvimento de softwares que significa dividir o código em partes menores.

Isso é uma técnica muito usada pois melhora muito o processo de manter o código.

As vantagens de modularizar um código são: é mais fácil de encontrar uma parte específica do código; é mais fácil de corrigir e atualizar o código; é possível reusar partes do seu código copiando um arquivo ao invés de ter que abrir o arquivo, procurar o que quer copiar e depois copiar.

A regra @import

Já vimos algumas regras de CSS: @media e @keyframes. A regra @import é a regra usada para modularizar o código CSS.

A regra @import permite importar um arquivo css dentro de outro arquivo css.

A regra @import deve estar no topo do arquivo CSS.

A regra @import também suporta seletores de mídia.

Vamos ver isso com um exemplo. Primeiramente crie um arquivo index.html e inclua isso no corpo dele:

                
                    <div class="vermelho"></div>
                    <div class="verde"></div>
                    <div class="azul"></div>
                    <div class="azul"></div>
                    <div class="verde"></div>
                    <div class="vermelho"></div>

Agora crie 4 arquvos CSS, dessa forma:

No arquivo div.css

                
                    div {
                        height: 100px;
                    }

No arquivo vermelho.css

                
                    .vermelho {
                        background-color: red;
                    }

No arquivo verde.css

                
                    .verde {
                        background-color: green;
                    }

No arquivo azul.css

                
                    .azul {
                        background-color: blue;
                    }

Agora crie um arquivo style.css e lembre-se de usar a tag link no arquivo index.html para que o arquivo style.css seja carregado na página

No arquivo style.css

                
                    @import "div.css";
                    @import url("vermelho.css");
                    @import "verde.css" screen and (max-width: 600px);
                    @import url("azul.css"); screen and (min-width: 600px);

Repare que existem dois jeitos de escrever o nome do arquivo a ser importado: "nome_do_arquivo"; ou url("nome_do_arquivo.css").

Repere também que os seletores de mídia funcionam!
