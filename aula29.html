String

    Vamos aprender algumas propriedades e métodos de String, caso queira aprender mais métodos de string, clique aqui.

    Concatenando strings com com `

    Já vimos como concatenar strings com aspas e aspas duplas:

                    
                    let nome = "Douglas";

                    console.log("Olá " + nome + ", bem-vindo!");
                    console.log('Oi ' + nome + ', bem-vindo!');

    Porém também podemos declarar strings com acento agudo: `Uma String`. Mas para concatenar strings desse tipo com variáveis, é um pouco diferente. Veja o exemplo: 

                    
                    let nome = "Douglas";

                    console.log(`Olá ${nome}, bem-vindo!`);

    Strings como Arrays

    String é uma cadeia de caracteres, portanto podemos pensar nela como um array de caracteres. De fato strings possuem propriedades semelhantes aos arrays. Vamos ver algumas delas agora:

    length

    Assim como arrays, strings possuem a propriedade length que retorna a quantidade de caracteres da string. Veja o exemplo: 

                    
                    "Olá".length;
                    "Olá Douglas".length;
                    " ".length;
                    "".length;
                    " Oi ".length;

    Acessando caracteres

    Assim como podemos acesar um elemento de um array, podemos acessar um caracter de uma string, veja o exemplo: 

                    
                    let nome = "Douglas Mariano Valero";

                    nome[0];
                    nome[1];
                    nome[7];
                    nome[14];

    Porém diferente de arrays, não podemos mudar um caractere de uma string diretamente, veja o exemplo: 

                    
                    let nome = "Carina";

                    nome[0] = "K"; //não funciona

                    console.log(nome);

    Split e Join

    Como strings são semelhantes a um array, podemos transformá-las em arrays e depois transformar de volta para strings. A utilidade disso é que assim podemos usar alguns métodos exclusivos de arrays nas strings convertidas e depois convertê-las de volta para string.

    O método split() recebe um separador como parâmetro e retorna um array com as substrings separadas. Caso queira separar caracter por caracter use o separador vazio "". Veja o exemplo: 

                    
                    let nome = "Douglas Mariano Valero";
                    let arrNome = nome.split(" ");

                    let texto = "Olá Mundo";
                    ler arrTexto  texto.split("");
                    console.log(arrTexto);

    O método join() junta um array de strings, usando um separador, e retorna uma única string. Veja o exemplo: 

                    
                    let arr = ["Douglas", "Mariano", "Valero"];
                    let nome = arr.join(" ");
                    console.log(nome);

    toLowerCase() e toUpperCase()

    O método toLowerCase() transforma uma string toda para letras minúsculas. E o método toUpperCase() para letras maiúsculas. 

                    
                    let nome = "Douglas";

                    nome = nome.toLowerCase();
                    console.log(nome);

                    nome = nome.toUpperCase();
                    console.log(nome);

    indexOf()

    O método indexOf() é usado para procurar uma substring dentro de uma string. Ele retorna -1 caso não encontre a substring, ou retorna a posição inicial a qual foi encontrada a substring. Veja os exemplos. 

                    
                    let nome = "Douglas Mariano Valero";

                    nome.indexOf("Mariano");
                    nome.indexOf("Silva");
                    nome.indexOf("v");
                    nome.indexOf("V");
                    nome.indexOf(" ");
                    nome.indexOf("w");

                    if(nome.indexOf("Valero") != -1){
                        console.log("Faz parte da família Valero!")
                    }

    substring()

    O método substring() retorna uma substring de uma string. Esse método recebe dois parâmetros, o primeiro é o índice do primeiro caractere, e o segundo é o comprimento da substring desejada, veja o exemplo: 

                    
                    let nome = "Douglas Mariano Valero";

                    let primeiroNome = nome.substring(0, 7);

                    console.log(primeiroNome);
