Aula 24 - Meta tags

As meta tags são tags HTML que tem informações sobre o próprio documento HTML. Já vimos algumas delas, agora vamos revê-las e aprender algumas bovas.

<!DOCTYPE html>

Essa tag, que geralmente está na primeira linha do documento HTML, serve para o navegador entender que este é um documento HTML.

<meta charset="utf-8">

Essa tag serve para determinar o tipo de caracteres da página. No nosso caso sempre será UTF-8. Existem outros tipos de caracteres, por exemplo para letras/símbolos chinesas ou árabes. É muito importante usar essa tag, caso contrário o seu site pode não funcionar direito para pessoas que usam outros tipos de caracteres como padrão.

<meta http-equiv="x-ua-compatible" content="ie=edge">

Essa tag também é muito importante pois faz o navegador Internet Explorar usar o mecanismo de renderização mais novo. Caso contrário seu site pode ter problemas de renderização no IE.

<meta name="viewport" content="width=device-width, initial-scale=1, viewport-fit=cover">

Essa meta tag é muito importante para que os navegadores determinem o campo de visão da tela do dispositivo. Caso contrário seu site pode ter problemas principalmente em smartphones.

<meta name="description" content="Descrição da página com menos de 150 caracteres">

Essa meta tag define uma descrição para o site e não pode ter mais de 150 caracteres.

<meta name="keywords" content="Palavras Chave">

Essa meta tag serve para determinar algumas palavras chave. Útil para mecanismos de busca.

<meta name="author" content="John Doe">

Essa meta tag serve para determinar o autor da página.

<meta http-equiv="refresh" content="10">

Essa meta tag serve para atualizar a página automaticamente. O valor em content é o número de segundos entre cada atualização automática.

<title>Título da página</title>

A tag title é importante para mecanismos de busca (Google). É recomendado que o título tenha menos de 55 caracteres, mais do que isso pode ser cortado.

<link rel="icon" type="image/png" href="https://github.githubassets.com/favicon.ico">

Essa tag define um ícone (imagem) que aparece na aba do navegador.

<html lang="pt-br">

Outra coisa importante é usar o atributo lang na tag html. No nosso caso usamos lang="pt-br" pois o conteúdo do nosso site é em português.
