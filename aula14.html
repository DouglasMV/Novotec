Vejamos agora essas propriedades e como usá-las:

colum-count

A propriedade column-count especifica o número de colunas em que o texto de um elemento deve ser dividido. Veja o exemplo abaixo:

No arquivo style.css:

                
                    div {
                        column-count: 2;
                    }

No arquivo index.html:

                
                    <div>Escreva um texto longo aqui!</div>

No exemplo anterior escreva um texto longo dentro da div para facilitar a visualização das colunas. Uma dica: no Visual Studio Code, digite lorem100 e aperte tab (ou enter), isso cria um texto aleatório com 100 palavras.

Mude o número de colunas e veja o resultado.

column-gap

A propriedade column-gap define a distância entre as colunas. No exemplo anterior adicione a propriedade column-gap: 40px; na div. Mude o valor e veja o que acontece.

column-rule

A propriedade column-rule cria uma divisória entre as colunas e define o estilo, a espessura e a cor dessa divisória. Muito parecida com a propriedade border que já conhecemos.

No exemplo anterior adicione a propriedade column-rule: 1px solid black; a div. Mude os valores, veja o que acontece.

column-span

A propriedade column-span pode fazer um elemento ocupar todas colunas. Essa propriedade não é usada no elemento que define as colunas, e sim nos elementos filhos dele. Veja o exemplo a seguir para esclarecer isso:

No arquivo style.css:

                
                    div {
                        column-count: 4;
                    }
                    h2 {
                        column-span: all;
                    }

No arquivo index.html:

                
                    <div>
                        <h2>Um título que ocupa todas colunas!</h2>
                        <p>
                           Um texto longo aqui!
                        </p>
                    </div>

Infelizmente existem dois problemas com essa propriedade:

O primeiro é que assim como várias propriedades de CSS mais novas, essa propriedade não funciona em todos navegadores. Column-span por exemplo não funciona no navegar Mozilla Firefox.

Outro problema é que os únicos valores possíveis para essa propriedade são: all e none. All faz o elemento ocupar todas colunas e none faz ele ocupar apenas uma. Seria bom se pudessemos escolher exatamente quantas colunas o elemento ocuparia, por exemplo ocupar 2 de 4 colunas, mas não é possível.

column-width

A propriedade column-width define a largura das colunas. Infelizmente o valor é único, ou seja, o mesmo para todas colunas. Não é possível dar um valor para cada coluna. No exemplo anterior adicione a propriedade column-width: 100px; na div. Altere o valor e veja o que muda.

O próximo exemplo mostra uma técnica de design responsivo (o layout muda de acordo com o tamanho da tela, estudaremos isso em breve) usando colunas. Usando um valor fixo para o tamanho das colunas, 200px por exemplo, e usando o column-count: auto; o número de colunas muda de acordo com o tamanho da tela, veja:

No arquivo style.css:

                
                    div {
                        column-rule: 3px solid purple;
                        column-count: auto;
                        column-width: 200px; 
                    }

No arquivo index.html:

                
                    <div>
                        Um texto longo aqui!
                    </div>

Mude o tamanho da largura da janela do seu navegador e observe o número de colunas.

Em aulas futuras veremos outras maneiras de criar layouts responsivos. Essas outras maneiras em geral são melhores que colunas, porém colunas tem uma característica única que pode ser usada como vantagem: colunas apenas afetam o texto, se colocarmos imagens ou outros tipos de elementos eles não serão afetados pelas colunas. Isso pode ser o que queremos em alguma situação específica, por isso é bom ter essa carta na manga.
