Aula 5 - CSS Cores

Cores

Cores em CSS são usadas para mudar a cor do texto, a cor do fundo, a cor da borda e outroas caracteristicas dos elementos.

Já vimos, por exemplo, como mudar a cor de fundo de um elemento, usando a propriedade background-color, seguida do nome de uma cor, veja abaixo.

                
                    p {
                        background-color: red;
                    }

Porém existem outas tipos de valores de cor, que estudaremos agora.

Cores RGB

Uma maneira de representar os valores de cores é utilizar valores numéricos RGB (Red, Green, Blue). Esse valor consiste em 3 valores numéricos que representam a quantidade de cores Vermelha, Verde e Azul respectivamente. Esses valores vão de 0 à 255, e a mistura dos 3 cria uma cor. Vejamos os exemplos abaixo.

                
                    h1 {
                        background-color: rgb(255, 0, 0);
                    }
                    h2 {
                        background-color: rgb(0, 255, 0);
                    }
                    h3 {
                        background-color: rgb(0, 0, 255);
                    }
                    h4 {
                        background-color: rgb(255, 255, 0);
                    }
                    h5 {
                        background-color: rgb(50, 50, 50);
                    }
                    h6 {
                        background-color: rgb(138, 110, 230);
                    }
Resultado:

h2

h3

h4

h5
h6

Repare que no h1 demos valor 255 para vermelho, 0 para verde e 0 para azul, resultando na cor vermelha. Semelhante nas tags h2 e h3 temos tudo verde e tudo azul respectivamente. Na tag h4 usamos 255 de vermelho, 255 de verde e 0 de azul, resultando em amarelo.

Na tag h5 usamos o valor 50 para as três cores, resultando em um cinza escuro. Sempre que os 3 valores são iguais temos um tom de cinza, sendo que quanto menor o valor mais escuro, e quanto maior o valor mais claro. Para obter a cor preta basta colocar os 3 valores como 0, e para obter a cor branca todos valores devem ser 255.

Na tag h6 usamos os valores rgb(138, 110, 230) criando uma cor diferente. Isso mostra como temos muito mais possibilidades de criar cores dessa maneira em comparação com os valores de nome de cores. Com valores RGB podemos criar mais de 16 milhões de cores diferentes, enquanto cores com nome (red, blue, yellow, orange, etc) existem menos de 150.

Crie um elemento e de a ele a propriedade background-color, use valores RGB, mude os 3 valores (Vermelho, Verde, Azul) usando números de 0 à 255, tente descobrir novas cores.

Cores Hexadecimais

Outra maneira de dar valores as cores é utilizando valores hexadecimais. Essa é a maneira mais usada, pois é mais curta e é o padrão de alguns programas de edição de imagens. É semelhante ao RGB, porém aqui os valores de cada cor vão de 00 até FF. Sim podemos usar números (de 0 a 9) e letras (A, B, C, D, E, F) sendo que cada letra tem um valor. O A vale 10, B vale 11, e assim por diante, vejamos:

Letra Valor
A 10
B 11
C 12
D 13
E 14
F 15

Cada cor (Vermelho, Verde e Azul) é representada por dois dígitos (números ou letras). Para representar a combinação das três cores usamos uma hashtag seguida de 6 dígitos, onde os dois primeiros representam a quantidade de vermelho, os dois do meio a quantidade de verde e os dois últimos a quantidade de azul. Por exemplo, #FF0000 tem o valor máximo de vermelho (pois F vale 15) e 0 para verde e azul, resultando na cor vermelha.

Vamos escrever o mesmo exemplo usado em RGB mas usando cores hexadecimais:

                
                    h1 {
                        background-color: #FF0000;
                    }
                    h2 {
                        background-color: #00FF00;
                    }
                    h3 {
                        background-color: #0000FF;
                    }
                    h4 {
                        background-color: #FFFF00;
                    }
                    h5 {
                        background-color: #323232;
                    }
                    h6 {
                        background-color: #8A6EE6;
                    }

Crie um elemento e de a ele a propriedade background-color, use valores hexadecimais, mude os valores usando valores de 0 à F, tente descobrir novas cores.

Mais sobre Cores

Existe mais uma maneira de criar cores em CSS, chamada HSL (Hue, Saturation, Lightness). Não estudaremos aqui pois não é muito utilizada, mas pode ser útil em algumas situações. Pesquise sobre essa maneira de representar cores para ampliar seus conhecimentos.

Outro elemento de cores é a opacidade, quando usamos valores RGB, podemos incluir um quarto valor que vai de 0 a 100 e representa a opacidade da cor, ou seja, o quanto ela é visível. Podemos usar isso para criar elementos semi-transparentes, ou seja, que é possível ver o que há atrás deles. Pesquise sobre isso.

Exercícios

Meu animal favorito

Faça uma lista não ordenada com o nome de pelo menos 10 animais diferentes.

Deixe todos nomes da lista com fonte tamanho 20px.

Escolha um animal favorito e deixe o nome dele com uma cor verde escura, e a cor de fundo do nome dele com um azul bem claro.

Cores e Tamanhos das Frutas

Faça uma lista não ordenada com pelo menos 20 frutas.

Crie 3 classes: pequena, media e grande. Atribua uma classe para cada fruta de acordo com o tamanho delas.

Na classe pequena utilize fonte de tamanho 16px.

Na classe media utilize fonte de tamanho 24px.

Na classe grande utilize fonte de tamanho 32px.

Mude a cor do fundo de cada fruta de acordo com a cor da fruta, todas devem ter cores diferente, utilize cores RGB ou hexadecimais.

Nascimento e Altura

Faça uma tabela com 3 colunas: Nome, Data de Nascimento, Altura.

Preencha a tabela com pelo menos 5 linhas.

Para as pessoas fazem aniversário nos meses de Janeiro até Junho, deixe a linha inteira (as 3 colunas) dela com uma cor de fundo amarela.

Para as pessoas fazem aniversário nos meses de Julho até Dezembro, deixe a linha inteira (as 3 colunas) dela com uma cor de fundo azul claro.

Para as pessoas que tem mais de 1,60m de altura deixe a cor do texto da linha toda (as 3 colunas) em verde escuro.

Para as pessoas que tem menos de 1,60m de altura deixe a cor do texto da linha toda (as 3 colunas) em vermelho escuro.

Deixe a tabela com uma borda de 1px sólida e preta em todas células. Sem espaços entre as bordas das células.

Questões para casa

1 - Explique a diferença em um elemento HTML auto-fechável e um elemento que possui uma tag de abertura e uma tag de fechamento.

Resposta: um elemento auto-fechável possui apenas a tag de abertura e seus atributos. Um elemento com tag de abertura e de fechamento, além dessas duas tags possui um contéudo (texto) entre as tags, e também pode possuir atributos na tag de abertura.

2 - Dê 3 exemplos diferentes de elementos HTML auto-fecháveis.

Resposta:

  • <!DOCTYPE html>
  • <img src="./img/imagem.jpg" alt="minha imagem">
  • <link rel="stylesheet" href="style.css">

3 - Qual elemento HTML representa o cabeçalho do documento HTML? Qual a função do cabeçalho do documento HTML? Dê 2 exemplos de elementos que podem ser filhos do cabeçalho.

Resposta: <head> é o elemento que representa o cabeçalho do documento HTML. A função dele é conter metadados, ou seja dados sobre o próprio documento. Exemplos: <title> <link> <meta>

4 - Qual elemento HTML representa o corpo do documento HTML? Qual a função do corpo do documento HTML?

Resposta: <body> representa o corpo do documento HTML. A função dele é envolver todo conteúdo visível da página.

5 - Quantos e quais são os elementos usados para criar cabeçalhos no corpo do documento HTML?

Resposta: São 6: <h1> <h2> <h3> <h4> <h5> <h6>

6 - O que são atributos de um elemento HTML? Dê um exemplo de elemento HTML com atributos.

Resposta: São informações extras sobre o elemento, que podem ser usadas para alterá-los de alguma forma. Exemplo: <a href="google.com" target="_blank">Clique aqui</a>

7 - Qual elemento HTML é usado para criar uma tabela? E para criar uma linha na tabela? E para criar uma célula de cabeçalho da tabela? E para criar uma célula normal da tabela?

Resposta: Para criar uma tabela usamos <table>. Para criar uma linha na tabela usamos <tr>. Para criar uma célula de cabeçalho usamos <th>. Para criar uma célula normal usamos <td>

.

8 - Explique as 3 maneiras de incluir CSS.

Resposta: Inline: os estilos são escritos como atributo style de uma tag. Interno: os estilos são escritos dentro da tag <style> que é filha do cabeçalho <head>. Externo: os estilos são escritos em um documento externo com a extensão .css, e esse documento deve ser linkado ao documento HTML com a tag <link>.

9 - Explique os 4 tipos de seletores CSS vistos em aula.

Resposta: Seletor de tag: seleciona todos elementos com a tag usada. (exemplo: p {}); Seletor de id: seleciona o elemento com o id usado. (exemplo: #meuId {}); Seletor de classe: seleciona todos elementos que possuem aquela classe. (exemplo .minhaClasse {}); Seletor universal: seleciona todos elementos. (exemplo: * {})

10 - Como fazer uma tabela ter bordas sólidas de 1px da cor preta em volta de todas células, sem espaço entre as bordas de uma célula para outra?

Resposta: Basta usar os seguinte estilos:

                
                    table {
                        border-collapse: collapse;
                    }
                    td, th {
                        border: 1px solid black;
                    }
