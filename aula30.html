Aula 30 - Array

    Vamos aprender propriedades e métodos de arrays, para ter uma lista completa clique aqui.

    Cuidado com length e []

    Vimos que um array tem a propriedade length, que teoricamente é a quantidade de elementos do array. Porém em JavaScript é possível mudar essa propriedade e isso pode dar resultados não desejados. Veja o exemplo: 

                
                let cores = ["laranja", "verde", "vermelho", "azul"];

                console.log(cores.length);

                cores.length = 2;
                console.log(cores);

                cores.length = 4;
                console.log(cores);

    Observe como mudar a propriedade length pode excluir valores do array, ou criar espaços vazios. Por isso NUNCA mude a propriedade length diretamente.

    Outra coisa que pode causar problemas, e também nunca devemos fazer, é modificar elementos do array que não existem. Veja o exemplo: 

                
                let cores = ["laranja", "verde", "vermelho", "azul"];
                cores[9] = "amarelo";

                console.log(cores);
                console.log(cores.length);

    Adicionando e Removendo Elementos

    O método push() adiciona um elemento no final de um array.

    O método pop() remove um elemento do final de um array.

    O método unshift() adiciona um elemento no início de um array.

    O método shift() adiciona um elemento no início de um array. 

                
                let cores = ["laranja", "verde", "vermelho", "azul"];

                cores.push("amarelo");
                console.log(cores);

                cores.pop();
                console.log(cores);

                cores.unshift("rosa");
                console.log(cores);

                cores.shift();
                console.log(cores);

    slice()

    O método slice() cria um subarray a partir de um array, ele recebe dois parâmetros, o primeiro é o início e o segundo é o final. Caso use apenas um parâmetro o array é copiado daquele índice até o final. Veja o exemplo: 

                
                let arr = ["zero", "um", "dois", "três", "quatro"];

                let arr1 = arr.slice(1, 3);
                console.log(arr1);

                let arr2 = arr.slice(2);
                console.log(arr2);

    Copiar um array

    Se você tentar copiar um array usando apenas o operador de atribuição (=), você vai perceber que não está criando um novo array, e sim uma outra referência para o mesmo array. Veja o exemplo: 

                
                let arr1 = [10, 20, 30];
                let arr2 = arr1;
                arr2[0] = 99;
                console.log(arr1);
                console.log(arr2);

    Para isso podemos usar o métod slice() sem nenhum parâmetro, assim criamos uma cópia independente do array, ou seja, podemos mudar a cópia sem mudar o array original. 

                
                let arr1 = [10, 20, 30];
                let arr2 = arr1.slice();
                arr2[0] = 99;
                console.log(arr1);
                console.log(arr2);

    concat()

    O método concat(), concatena (junta) um array com outros elementos, que podem ser valores ou outros arrays. Veja: 

                
                let arr = [10, 20, 30];
                
                arr.concat(40);
                console.log(arr);

                arr.concat(50, 60, 70);
                console.log(arr);

                arr.concat([80, 90, 100]);
                console.log(arr);

                arr.concat([110, 120, 130, 140], 150, 160, [170, 180]);
                console.log(arr);

    filter()

    O método filter recebe uma função como parâmetro que retorna um boolean (true ou false), e executa essa função para cada item do array, e cria um novo array com os elementos que retornaram true. Veja o exemplo: 

                
                let arr = [-1, -2, 0, 3, 10, -4, 7];

                let arrFiltrado = arr.filter(function(item){
                    return item >= 0;
                });

                console.log(arrFiltrado);

    map()

    O método map() recebe uma função como parâmetro, executa a função para cada item do array e cria um novo array com os valores retornados pela função para cada item. Veja o exemplo: 

                
                let arr = [-1, -2, 0, 3, 10, -4, 7];

                let dobro = arr.map(function(item){
                    return item*2;
                });

                console.log(dobro);

    sort()

    O método sort() ordena os elementos do array. Por padrão ordena na ordem alfabbética. 

                
                let cores = ["laranja", "verde", "vermelho", "azul"];
                cores.sort();
                console.log(cores);

    Para ordenar de outra maneira podemos usar uma função como parâmetro. Para entender melhor essa função clique aqui.

    Aqui vamos mostrar um exemplo muito usado, que é para ordenar números do maior para o menos: 

                
                let arr = [-1, -2, 0, 3, 10, -4, 7];
                arr.sort(function(a, b) {
                    return a-b;
                });
                console.log(arr);
