Aula 22 - Grid

Na aula passada aprendemos sobre Flexbox, que é uma ótima ferramenta de CSS para organizar a exibição de elementos em uma direção. Grid tem a capacidade de posicionar elementos nas duas direções (horizontal e vertical) ao mesmo tempo. Além disso nos dá um controle maior e muitas opções que Flexbox não tem.

Criando uma Grid

Semelhante ao Flexbox, para criar uma Grid é preciso ter um container e os filhos desse container serão os itens da Grid.

No arquivo style.css:

                
                    #meu-container {
                        display: grid;
                        background-color: magenta;
                    }
                    .item {
                        background-color: lightblue;
                        border: 1px solid black;
                    }

No arquivo index.html:

                
                    <div id="meu-container">
                        <div class="item">A</div>
                        <div class="item">B</div>
                        <div class="item">C</div>
                        <div class="item">D</div>
                        <div class="item">E</div>
                    </div>

grid-template-columns (container)

A propriedade grid-template-columns serve para definir o tamanho e a quantidade de colunas de uma Grid. Para isso basta escrever os valores da largura de cada coluna em ordem (da primeira até a última) separados por espaço. A quantidade de valores escritos será a quantidade de colunas, e os valores serão a largura de cada coluna. Veja o exemplo:

No arquivo style.css:

                
                    #meu-container {
                        display: grid;
                        width: 800px;
                         grid-template-columns: 200px 150px 450px; 
                        background-color: magenta;
                    }
                    .item {
                        background-color: lightblue;
                        border: 1px solid black;
                    }

No arquivo index.html:

                
                    <div id="meu-container">
                        <div class="item">A</div>
                        <div class="item">B</div>
                        <div class="item">C</div>
                        <div class="item">D</div>
                        <div class="item">E</div>
                    </div>

Repare que criamos 3 colunas, porém temos 5 elementos, assim os 3 primeiros ocupam as 3 colunas da primeira linha e os outros 2 ocupam as 2 primeiras colunas da segunda linha. Ou seja, por padrão os elementos ocupam as céluas da esquerda para direita, de cima para baixo. Mas podemos mudar isso, e veremos como mais adiante.

grid-template-rows (container)

O grid-template-rows funciona semelhante ao grid-template-columns, porém para definir linhas. Veja o exemplo:

No arquivo style.css:

                
                    #meu-container {
                        display: grid;
                        width: 800px;
                        grid-template-columns: 200px 150px 450px;
                        grid-template-rows: 100px 150px 50px; 
                        background-color: magenta;
                    }
                    .item {
                        background-color: lightblue;
                        border: 1px solid black;
                    }

No arquivo index.html:

                
                    <div id="meu-container">
                        <div class="item">A</div>
                        <div class="item">B</div>
                        <div class="item">C</div>
                        <div class="item">D</div>
                        <div class="item">E</div>
                    </div>

Repare que agora temos 3 linhas e 3 colunas. O que nos dá um total de 9 células. Mas como só temos 5 items, a terceira célula da segunda linha e as 3 células da terceira linha não tem nenhum elemento.

grid-template (container)

A propriedade grid-template serve de abreviação para as propriedades grid-template-rows e grid-template-columns. Veja o exemplo:

No arquivo style.css:

                
                    #meu-container {
                        display: grid;
                        width: 800px;
                        grid-template: 100px 150px 50px / 200px 150px 450px; 
                        background-color: magenta;
                    }
                    .item {
                        background-color: lightblue;
                        border: 1px solid black;
                    }

No arquivo index.html:

                
                    <div id="meu-container">
                        <div class="item">A</div>
                        <div class="item">B</div>
                        <div class="item">C</div>
                        <div class="item">D</div>
                        <div class="item">E</div>
                    </div>

Repare que esse exemplo é o mesmo que o anterior, apenas escrito de forma diferente. Primeiro colocamos os valores das linhas (rows) e depois o das colunas (columns), separados por uma barra "/".

Perceba que nesse exemplo usamos valores absolutos (em px) para a largura das colunas, que somados dão exatamente o tamanho do container. Mude esses valores e perceba que as colunas podem ficar menores que o container ou sair para fora dele. Além disso se você usar valores de % ou vh, vw, é bem provável que isso ocorra também. E o fato do valor da largura (width) do próprio container ser absoluto, faz com que esse exemplo não seja responsivo.

Podemos consertar isso de uma forma: usando um valor de width relativo (vw, %) para o container e usando apenas valores de % nas colunas.

Essa não é uma solução ruim, porém com Grid existe uma solução muito melhor, que nos permite inclusive usar valores absolutos (px) e manter a responsividade. Isso é possível usando um novo valor relativo introduzido em Grid: frações. Que veremos agora.

Frações (container)

Frações (fr) é uma unidade relativa introduzida ao CSS especialmente para Grid. Ela pode ser usada tanto para largura (colunas) quanto para altura (linhas). É muito mais fácil entendê-la utilizando exemplos:

No arquivo style.css:

                
                    #meu-container {
                        display: grid;
                        width: 100vw;
                        grid-template: 100px 150px 50px / 2fr 1fr 4fr; 
                        background-color: magenta;
                    }
                    .item {
                        background-color: lightblue;
                        border: 1px solid black;
                    }

No arquivo index.html:

                
                    <div id="meu-container">
                        <div class="item">A</div>
                        <div class="item">B</div>
                        <div class="item">C</div>
                        <div class="item">D</div>
                        <div class="item">E</div>
                    </div>

Repare que o container agora não tem largura absoluta e sim relativa. Nesse exemplo usamos fr nas colunas (2fr 1fr 4fr). Assim a primeira coluna (2fr) sempre terá o dobro de tamanho da segunda (1fr), e a terceira coluna (4fr) será 4 vezes maior que a segunda coluna (1fr). Aumente e diminua a janela do navegador e veja como agora o layout é responsivo.

O próximo exemplo mostra uma vantagem muito boa de se usar fr:

No arquivo style.css:

                
                    #meu-container {
                        display: grid;
                        width: 100vw;
                        grid-template: 100px 150px 50px / 200px 1fr 1fr; 
                        background-color: magenta;
                    }
                    .item {
                        background-color: lightblue;
                        border: 1px solid black;
                    }

No arquivo index.html:

                
                    <div id="meu-container">
                        <div class="item">A</div>
                        <div class="item">B</div>
                        <div class="item">C</div>
                        <div class="item">D</div>
                        <div class="item">E</div>
                    </div>

Nesse exemplo usamos um valor absoluto em uma das colunas (200px). E nas outras colunas usamos 1fr e 1fr. Ou seja, não importa o tamanho da tela a primeira coluna sempre terá 200px de largura. E o restante do tamanho da tela será dividido igualmente entre a segunda e a terceira colunas. As vantagens disso são que o layout é responsivo e os item da grid nunca ultrapassam ou deixam sobras espaço do container. Tente fazer isso usando % ou vw, é impossível. Temos apenas que tomar cuidado para não usarmos valores absolutos (px) muito grandes, pois isso pode causar a indesejada barra de rolagem horizontal em dispositivos menores.

Tente mudar os valores e as unidades, criar mais coluna, etc. Descubra o que é possível criar com Grid usando fr. e lembre-se que isso também pode ser usado nas linhas (rows).

repeat (container)

As propriedades que definem o número e o tamanho das linhas e colunas podem receber um valor especial: repeat. Que serve para criar um certo número de linhas ou colunas repetidos (de mesmo tamanho)

Repeat recbe dois valores entre parenteses, o primeiro é o número de colunas ou linhas, e o segundo valor é o tamanho. Veja o exemplo:

No arquivo style.css:

                
                    #meu-container {
                        display: grid;
                        width: 800px;
                        grid-template-columns:  repeat(3, 250px)  50px;
                        grid-template-rows:  repeat(2, 300px); 
                        background-color: magenta;
                    }
                    .item {
                        background-color: lightblue;
                        border: 1px solid black;
                    }

No arquivo index.html:

                
                    <div id="meu-container">
                        <div class="item">A</div>
                        <div class="item">B</div>
                        <div class="item">C</div>
                        <div class="item">D</div>
                        <div class="item">E</div>
                    </div>

Isso facilita a criação de grids com muitas linhas e colunas repetidas. Lembre-se que o valor do tamanho das linhas ou colunas pode ser em frações (fr) também.

minmax (container)

Outro valor especial que pode ser usado para definir o tamanho das linas e colunas é minmax. Ele determina um valor mínimo e um máximo. Isso é bom pois garante que se a tela for muito pequena a coluna terá um tamanho mínimo, ou seja, não ficará menor que aquilo. E do mesmo modo se a tela for muito grande a coluna terá um tamanho máximo, ou seja, não ficará maior que aquilo. Veja o exemplo:

No arquivo style.css:

                
                    #meu-container {
                        display: grid;
                        width: 100vw;
                        grid-template: 100px 150px 50px / 100px minmax(100px, 300px);
                        background-color: magenta;
                    }
                    .item {
                        background-color: lightblue;
                        border: 1px solid black;
                    }

No arquivo index.html:

                
                    <div id="meu-container">
                        <div class="item">A</div>
                        <div class="item">B</div>
                        <div class="item">C</div>
                        <div class="item">D</div>
                        <div class="item">E</div>
                    </div>

Algo importante aqui é o fato de que a unidade fr só pode ser usada para valores máximos, nunca para mínimos. (mas existe a possibilidade disso mudar no futuro)

grid-gap (container)

É possível criar um espaço entre as linhas, ou entre as colunas da Grid. Para isso usamaos a propriedade grip-gap. Ela recebe dois valores, o primeiro é o espaço entre as linhas (vertical) e o segundo o espaço entre as colunas (horizontal). Veja o exemplo:

No arquivo style.css:

                
                    #meu-container {
                        display: grid;
                        width: 100vw;
                        grid-template: repeat(3, 200px) / repeat(2, 1fr);
                        grid-gap: 50px 10px;
                        background-color: magenta;
                    }
                    .item {
                        background-color: lightblue;
                        border: 1px solid black;
                    }

No arquivo index.html:

                
                    <div id="meu-container">
                        <div class="item">A</div>
                        <div class="item">B</div>
                        <div class="item">C</div>
                        <div class="item">D</div>
                        <div class="item">E</div>
                    </div>

Repare que agora existe um espaço entre as linhas e entre as colunas. Também é possível usar as propriedades grid-row-gap e gris-column-gap para mudar esses valores individualmente, porém na prática quando queremos criar um gap geralmente queremos em ambas direções, por isso grid-gap é mais usada.

Grid Itens

Até agora apenas vimos propriedades para o container grid, ou seja, o elemento pai. E os itens, os elementos filhos, ocupavam exatamente uma célula da Grid, na ordem em que estão escritos no documento HTML. Podemos mudar isso, podemos fazer um elemento ocupar várias linhas e colunas. Mas primeiro precisamos aprender como "contar" as linhas e colunas de uma Grid.

Observe a imagem abaixo, ela representa uma Grid com 3 colunas e 3 linhas. Porém note que temos uma numeração que vai de 1 a 4 nas bordas das colunas e linhas. Essa numeração que é usada para definir onde cada elemento será posicionado.

grid-row (itens)

Usando a propriedade grid-row, em um item da grid, é possível fazer um elemento ocupar multiplas linhas. Veja o exemplo:

No arquivo style.css:

                
                    #meu-container {
                        display: grid;
                        width: 620px;
                        grid-template: repeat(3, 200px) / repeat(3, 200px);
                        grid-gap: 10px 10px;
                        background-color: magenta;
                    }
                    .item {
                        background-color: lightblue;
                        border: 1px solid black;
                    }
                    #div1 {
                        grid-row: 1 / 3;
                    }

No arquivo index.html:

                
                    <div id="meu-container">
                        <div id="div1" class="item">A</div>
                        <div class="item">B</div>
                        <div class="item">C</div>
                        <div class="item">D</div>
                        <div class="item">E</div>
                    </div>

Veja que a div1 ocupa agora 2 linhas, lembre-se da numeração que vimos, 1 a 3 ocupa as duas primeiras linhas. Mude os valores e veja o resultado.

grid-column (itens)

Usando a propriedade grid-column, em um item da grid, é possível fazer um elemento ocupar multiplas colunas. Veja o exemplo:

No arquivo style.css:

                
                    #meu-container {
                        display: grid;
                        width: 620px;
                        grid-template: repeat(3, 200px) / repeat(3, 200px);
                        grid-gap: 10px 10px;
                        background-color: magenta;
                    }
                    .item {
                        background-color: lightblue;
                        border: 1px solid black;
                    }
                    #div1 {
                        grid-column: 2 / 4;
                    }

No arquivo index.html:

                
                    <div id="meu-container">
                        <div id="div1" class="item">A</div>
                        <div class="item">B</div>
                        <div class="item">C</div>
                        <div class="item">D</div>
                        <div class="item">E</div>
                    </div>

Veja que a div1 ocupa agora 2 colunas, lembre-se da numeração que vimos, 2 a 4 ocupa a segunda e a terceira colunas. Mude os valores e veja o resultado.

grid-area (itens)

A propriedade grid-area é uma abreviação para as propriedades grid-row e grid-column. Ela recebe 4 valores separados por "/", a ordem dos valores é: o começo da linha, o começa da coluna, o fim da linha, o fim da coluna. Veja o exemplo:

No arquivo style.css:

                
                    #meu-container {
                        display: grid;
                        width: 620px;
                        grid-template: repeat(3, 200px) / repeat(3, 200px);
                        grid-gap: 10px 10px;
                        background-color: magenta;
                    }
                    .item {
                        background-color: lightblue;
                        border: 1px solid black;
                    }
                    #div1 {
                        grid-area: 1 / 2 / 3 / 4;
                    }

No arquivo index.html:

                
                    <div id="meu-container">
                        <div id="div1" class="item">A</div>
                        <div class="item">B</div>
                        <div class="item">C</div>
                        <div class="item">D</div>
                        <div class="item">E</div>
                    </div>
