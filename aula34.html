Aula 34 - setTimeout e setInterval

    setTimeout

    A função setTimeout recebe dois argumentos: uma função; e um valor numérico.

    O que acontece é que a função passada como argumento é executada após um tempo. Esse tempo é o valor numérico passado como argumento, em milissegundos. Veja o exemplo:

    No index.html:

              
            <h1 id="cabecalho">Cabeçalho!</h1>

            <button id="btn">Clique aqui!</button>

    No app.js:

              
            let cab = document.getElementById("cabecalho");
            let btn = document.getElementById("btn");
            
            btn.addEventListener('click', function () {
              setTimeout(function () {
                cab.remove();
              }, 1000);
            });

    clearTimeout

    Quando declaramos uma função do tipo setTimeout, podemos salvá-la em uma variável, e podemos usar essa variável como parâmetro da função clearTimeout para cancelar o temporizador. Veja o exemplo: 

              
            <h1 id="cabecalho">Cabeçalho!</h1>

            <button id="btn">Não apagar!!</button>

    No app.js:

              
            let cab = document.getElementById("cabecalho");
            let btn = document.getElementById("btn");
            
            let apagar = setTimeout(function () {
              cab.remove();
            }, 1000);
            
            btn.addEventListener('click', function () {
              clearTimeout(apagar);
            });

    setInterval

    Essa função, semelhante a setTimeout, recebe uma função e um valor em milissegundos como parâmetros. Essa função é executada a cada intervalo de tempo, veja o exemplo:

    No index.html:

              
            <h1>Segundos: <span id="segundos">0</span></h1>

    No app.js:

              
            let segundos = document.getElementById("segundos");
            let contador = 0;
            setInterval(function() {
              contador++;
              segundos.textContent = contador;
            }, 1000);

    clearInterval

    Semelhante a clearTimeout, podemos cancelar uma função do tipo setInterval usando clearInterval, veja o exemplo:

    No index.html:

              
            <h1>Segundos: <span id="segundos">0</span></h1>

            <button id="btn">Parar o tempo!</button>

    No app.js:

              
            let btn = document.getElementById("btn");
            let segundos = document.getElementById("segundos");
            let contador = 0;
            
            let tempo = setInterval(function () {
              contador++;
              segundos.textContent = contador;
            }, 1000);
            
            btn.addEventListener('click', function () {
              clearInterval(tempo);
            });
