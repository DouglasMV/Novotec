Seletores

Na aula 4 já vimos 4 tipos de seletores: tag, id, class e universal. Vamos relembrá-los rapidamente.

Tag

O seletor de Tag seleciona todos elementos declarados com aquela tag. Para usá-lo basta escrever o nome da tag e colocar as propriedas CSS entre colchetes:

                
                    p {
                        color: red;
                    }

Id

O seletor por Id utiliza o atributo id em uma tag HTML para selecionar aquele elemento. O atributo id deve ser único, ou seja, se existe um elemento com id="meuID", não pode existir outro elemento com o mesmo id.

Para selecionar um elemento por id, utilizamos a hashtag (#) seguida do id, veja o exemplo:

                
                    #paragrafo1 {
                        color: red;
                    }

Class

O seletor por Classe utiliza o atributo class em uma tag HTML para selecionar aquele elemento. O atributo class não precisa deve ser único, ou seja, podem existir vários elemento com class="minhaClasse".

Para selecionar um elemento por class, utilizamos o ponto (.) seguido do nome da classe, veja o exemplo:

                
                    .texto-vermelho {
                        color: red;
                    }

Universal

O seletor universal, como o próprio nome sugere, seleciona todos elementos HTML da página, independente de tag, clase ou id, sem exceção.

Para usar o seletor universal, utilizamos o asterístico (*), veja o exemplo:

                
                    * {
                        color: red;
                    }

Agora que já revimos esses seletore, vamos aprender outros.

Agrupando Seletores

É comum existirem situações que queremos usar uma mesma propriedade CSS em vários elementos diferentes, é claro que podemos criar uma classe e usá-la em todos elementos. Porém às vezes é mais fácil agrupar seletores, para isso basta escrever vários seletores separados por vírgula. Veja o exemplo abaixo, onde queremos selecionar todos elementos de cabeçalho (h1 à h6) e aplicar uma mesma cor de texto à eles:

p>No arquivo style.css:

                
                    h1, h2, h3, h4, h5, h6 {
                        color: red;
                    }

No arquivo index.html:

                
                    <h1>Olá do h1</h1>
                    <h2>Olá do h2</h2>
                    <h3>Olá do h3</h3>
                    <h4>Olá do h4</h4>
                    <h5>Olá do h5</h5>
                    <h6>Olá do h6</h6>

Combinando Seletores

Podemos combinar seletores, basta usar dois ou mais seletores escritos juntos (sem espaço), veja o exemplo:

No arquivo style.css:

                
                    p.red {
                        color: red;
                    }
                    p#paragrafo2 {
                        background-color: orange;
                    }
                    p.bg-purple{
                        background-color: purple;
                    }
                    p.red.bg-purple {
                        font-size: 40px;
                    }

No arquivo index.html:

                
                    <p>p sem class e sem id</p>
                    <p id="paragrafo2">p sem class e com id="paragrafo2"</p>
                    <p class="red">p com class="red"</p>
                    <p class="bg-purple">p com class="bg-purple"</p>
                    <p class="red bg-purple">p com class="red bg-purple"</p>
                    <div class="red bg-purple">div com class="red bg-purple"</div>

Nesse exemplo podemos reparar que: se quisermos usar duas ou mais classes em um elemento HTML basta separar o nome das classes com um espaço no atributo class (<p class="red bg-purple">); o seletor p.red seleciona todos parágrafos com a classe red; o seletor p#paragrafo2 seleciona o paragrafo com id paragrafo2, não muito útil já que o id deve ser único de qualquer maneira, ou seja só o seletor #paragrafo2 teria o mesmo resultado; o seletor p.bg-purple seleciona todos parágrafos com a classe bg-purple; o seletor p.red.bg-purple seleciona apenas os parágrafos que tem ambas classes red e bg-purple ao mesmo tempo; repare que a div com classes red e bg-purple não é selecionada por nenhum dos seletores usados, pois todos começam com p, ou seja, apenas selecionam parágrafos com aquelas classes, não selecionam nenhuma outra tag (div nesse exemplo) mesmo que tenham as classes usadas.

Seletor de Descendentes

O seletor de descendentes seleciona todos elementos que são descendentes de outro elemento. Basta escrever um seletor (tag, id ou class) do elemento "de fora", seguido de um espaço e depois um seletor do elemento "de dentro", o descendente. Veja o exemplo:

No arquivo style.css:

                
                    div p {
                        color: red;
                    }

                    div .fundo-azul {
                        background-color: blue;
                    }

No arquivo index.html:

                
                    <div>
                        <p>Sou descendente de div (filho)</p>
                        <ul>
                            <li>
                                <p class="fundo-azul">Sou descendente de div ("bisneto") e tenho class="fundo-azul"</p>
                            </li>
                        </ul>
                    </div>

                    <p class="fundo-azul">Não sou descendente de div e tenho class="fundo-azul"</p>

Repare que o descendente não precisa ser filho direto, pode ser filho do filho do filho...

Seletor de Filho

O seletor de filho é semelhante ao seletor de descendente, porém é necessário que o descendente seja filho direto do primeiro elemento. Basta escrever um seletor (tag, id ou class) do elemento pai, seguido de um sinal de maior (>) e depois um seletor do elemento filho. Veja o exemplo:

No arquivo style.css:

                
                    div>p {
                        color: red;
                    }

                    div>.fundo-azul {
                        background-color: blue;
                    }

No arquivo index.html:

                
                    <div>
                        <p>Sou descendente de div (filho)</p>
                        <ul>
                            <li>
                                <p class="fundo-azul">Sou descendente de div ("bisneto") e tenho class="fundo-azul"</p>
                            </li>
                        </ul>
                    </div>

                    <p class="fundo-azul">Não sou descendente de div e tenho class="fundo-azul"</p>

Repare que apenas o parágrafo (p) que é filho direto de uma div teve a cor do texto alterada. E nenhum parágrafo teve a cor de fundo alterada, pois nenhum elemento com classe="fundo-azul" é filho (descendente direto) de uma div.

Pseudo-class

Em css temos as chamadas pseudo-classes, elas correespondem a estados dos eslementos, por exemplo a pseudo-classe hover corresponde ao estado de um elemento quando o cursosr do mouse está sobre ele. Para usar uma pseudo-classe basta selecionar o elemento com um dos seletores já vistos, e depois usar dois ponto (:) e o nome da pseudo-classe. Veja o exemplo:

No arquivo style.css:

                
                    p:hover {
                        background-color: blue;
                    }

No arquivo index.html:

                
                    <p>Coloque o cursor do mouse em cima de mim!</p>

Passe o mouse por cima do paragrafo e veja o que acontece.

Um truque comum com a pseudo-classe hover é utilizá-la pra esconder/mostrar um elemento, alterando a propriedade display do mesmo. Veja o exemplo:

No arquivo style.css:

                
                    div {
                        height: 300px;
                        width: 300px;
                        background-color: yellow;
                    }
                    p {
                        display: none;
                    }
                    div:hover p{
                        display: block;
                    }

No arquivo index.html:

                
                    <div>
                        <p>Apareci!</p>
                    </div>
