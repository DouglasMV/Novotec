Aula 26 - Condicionais e Loops

Nessa aula aprenderemos sobre condicionais (ou controle de fluxo) e loops (ou repetição). Basicamente condicionais nos permitem executar um código apenas se uma condição é satisfeita. E loops nos permitem repetir uma operação várias vezes, sem ter que escrevê-la repetidamente.

Mas antes disso precisamos aprender sobre operadores de comparação e operadores lógicos, que são muito usados em condicionais e loops.

Operadores de comparação

Esse tipo de operador compara dois valores e retorna um boolean: true se a comparação for verdadeira; false se a comparação for falsa. Por exemplo, imagine que queremos saber se a idade de um usuário é maior que 18, basta comparar uma variável idade com o valor 18.

Existem 8 tipos de comparação em JavaScript:

  • < (menor que)
  • > (maior que)
  • <= (menor ou igual a)
  • >= (maior ou igual a)
  • == (igual a)
  • != (diferente de)
  • === (estritamente igual a)
  • !== (estritamente diferente de)

Aqui devemos prestar atenção em 3 coisas:

1 - Evitar strings nos 4 primeiros: apesar de ser possível compara strings com os operadores de desigualdade (<, >, <=, >=) não é recomendado (pelo menos por enquanto), pois ele não obedece exatamente a ordem alfabética quando misturamos letras maiúsculas e minúsculas. Por exemplo: "a" < "B" retorna false, quando provavelmente sua intuição diria que retornaria true. Uma solução para esse problema seria deixar todas letras das duas palavras minúsculas (ou todas maiúsculas), isso é possível porém ainda não resolve por exemplo o caso de palavras com acentos. Por isso evite usar esses operadores com strings, use apenas com numbers.

2 - igual converte, estritamente igual não converte:você deve estar se perguntando qual a diferença entre o igual (==) e o estritamente igual (===)? A diferença é que o igual (==) converte automaticamente os valores para o mesmo tipo, e depois compara. Já o estritamente igual não converte. Então por exemplo, 1 == "1" retorna true, enquanto 1 === "1" retorna false. O mesmo vale pra diferente (!=) e estritamente diferente (!==).

3 - Use estritamente igual sempre que possível: pelo fato do igual realizar a conversão, ele é um pouco mais lento, por isso use o estritamente igual (===) sempre que possível. Existem alguns casos em que usar o igual (==) pode ser mais vantajoso, porém na maioria dos casos o estritamente igual é melhor.

Vejamos alguns exemplos, digite essas expressões, uma de cada vez, no console de seu navegador, antes de apertar enter e ver o resultado tente adivinhar se é true ou false.

                
                    2 < 3
                    2 < 1
                    10 > 10
                    10 > 9
                    7 <= 7
                    7 >= 7
                    11 == "11"
                    11 === "11"
                    "laranja" === "laranja"
                    "banana" !== "laranja"
                    "uva" !== "uva"

                    let idadeDouglas = 28;
                    let idadeFulano = 17;
                    idadeDouglas >= 18;
                    idadeFulano >= 18;

Operadores Lógicos

Em JavaScript existem 3 operadores lógicos: não; e; ou. Vejamos como usar cada um deles.

! (não)

O primeiro operador lógico que vamos estudar é o não representado pelo ponto de exclamação (!). O que ela faz é simplesmente pegar um valor ou variável que esteja logo após ele, converte-o para um boolean se ainda não for e inverte o resultado. Ou seja, se o valor era true ele retorna false, se o valor era false ele retorna true. Veja os exemplos:

                
                    !true
                    !false
                    !0
                    !""
                    !"string não vazia"
                    !11
                    !(1 < 2)
                    !(1 > 2)
                    !("Oi" === "Oi")
                    !(10 === "10")

&& (e)

O próximo operador lógico que veremos é o e, representado por &&. Usamos ele entre dois valores ou expressões.

A maneira mais fácil de entender esse operador, (e que é a correta para outras linguagens de programação mas não exatamente para o JS) é a seguinte: se as duas expressões são verdadeiras retorna true, se uma das duas é falsa retorna false.

Isso está correto em partes, pois em JavaScript os operadores lógicos && e || (ou) não retornam um boolean, e sim uma das duas expressões usadas. Porém como na maior parte das vezes o valor retornado é convertido para um boolean, as vezes não percebemos isso. Leia a seguir o que é retornado na realidade em cada caso para o operador &&:

Se a primeira expressão for falsa ela é retornada e a segunda não é computada (é ignorada). Se a primeira expressão for verdadeira, retorna-se a segunda expressão. Tente compreender a lógica por trás disso.

Veja os exemplos:

                
                    "Olá" && "Oi"
                    "Olá" && ""
                    1 < 2 && 1 < 3
                    1 < 2 && 1 < 0
                    1 < 0 && 1 < 2
                    1 < 0 && 1 < -1

|| (ou)

O último operador lógico é o ou, representado por || (aperte shift e a tecla do lado esquerdo da letra Z duas vezes). Semelhante ao && usamos ele entre dois valores ou expressões.

A maneira mais fácil de entender esse operador, (e que é a correta para outras linguagens de programação mas não exatamente para o JS) é a seguinte: se uma das expressões é verdadeira retorna true, se as duas são falsas retorna false.

Isso está correto em partes, pois em JavaScript os operadores lógicos && e || (ou) não retornam um boolean, e sim uma das duas expressões usadas. Porém como na maior parte das vezes o valor retornado é convertido para um boolean, as vezes não percebemos isso. Leia a seguir o que é retornado na realidade em cada caso para o operador ||:

Se a primeira expressão for verdadeira ela é retornada e a segunda não é computada (é ignorada). Se a primeira expressão for falsa, retorna-se a segunda expressão. Tente compreender a lógica por trás disso.

Veja os exemplos:

                
                    "Olá" || "Oi"
                    "Olá" || ""
                    1 < 2 || 1 < 3
                    1 < 2 || 1 < 0
                    1 < 0 || 1 < 2
                    1 < 0 || 1 < -1

Além dos loops e condicionais que veremos a seguir, o operador || é muito útil em um caso específico: usar um valor padrão caso uma variável não exista (seja falsa). Veja o exemplo:

No arquivo app.js:

                
                    let nome;

                    nome = prompt("Digite seu nome:");

                    alert("Olá " + (nome || "visitante") + "!");

Execute o script acima e quando for solicitado o nome não digite nada, apenas aperter o OK, viu o que aconteceu? Por quê isso aconteceu?

Condicionais

Condicionais são operadores que executam um bloco de código apenas se uma determinada condição for satisfeita.

if

O primeiro condicional é o if, em português se. Ele recebe uma condição entre parenteses e executa o código entre chaves se a condição for verdadeira. Veja o exemplo:

No arquivo app.js:

                
                    let senha = prompt("Digite a senha super secreta:");

                    if (senha === "senha super secreta") {
                        alert("Login com sucesso!");
                    }

else

Geralmente quando usamos o if, e a condição não é satisfeita, queremos executar outro código. Você pode estar pensando: "é só escrever o outro código depois do bloco entre chaves do if". Sim, podemos fazer isso, mas o problema é que esse código será executado de qualquer maneira, seja a condição do if verdadeira ou falsa. Veja o exemplo:

No arquivo app.js:

                
                    let idade = prompt("Digite a sua idade:");

                    if (idade >= 18) {
                        alert("Pode entrar!");
                    }

                    alert("Não pode entrar!");

Perceba que a mensagem "Não pode entrar" é exibida de qualquer maneira, mas o que queremos é que ela apareça apenas quando a condição idade >= 18 for falsa. Para isso serve o else, em português senão:

No arquivo app.js:

                
                    let idade = prompt("Digite a sua idade:");

                    if (idade >= 18) {
                        alert("Pode entrar!");
                    } else {
                        alert("Não pode entrar!");
                    }

else if

Agora imagine que queremos dividir as pessoas em 3 grupos por idade: menores de 18; entre 18 e 60; e maiores de 60. Para isso podemos utilizar outro if na frente do else, veja:

No arquivo app.js:

                
                    let idade = prompt("Digite a sua idade:");

                    if (idade < 18) {
                        alert("Menor não pode entrar!");
                    } else if (idade < 60) {
                        alert("Adulto pode entrar!");
                    } else {
                        alert("Idoso pode entrar e tem preferência!")
                    }

Podemos usar quantos else if quisermos, porém quando temos muitos casos e os valores são específicos podemos usar outro condicional, o switch que veremos a seguir.

switch

O switch é um condicional que pode ser usado quando o valor a ser comparado deve ser estritamente igual (===) a algum valor. Por exemplo, imagine que perguntamos ao usuário qual é sua a cor favorita. E queremos mudar a cor de fundo da página de acordo com a cor digitada, veja o exemplo:

                
                        let corFavorita = prompt("Digite a sua cor favorita:");

                        switch(corFavorita) {
                            case "vermelho":
                                document.write('<body style="background-color: red"></body>');
                                break;
                            case "verde":
                                document.write('<body style="background-color: green"></body>');
                                break;
                            case "azul":
                            case "blue":
                                document.write('<body style="background-color: blue"></body>');
                                break;
                            default:
                                document.write('<body style="background-color: black"></body>');
                        }

Repare na sintaxe. Escrevemos switch e depois entre parenteses a variável que desejamos comparar com os valores. Depois usamos a palavra case seguida de um valor e de dois pontos. Na linha de baixo escrevemos o código a ser executado caso a variável tenha aquele valor. Repare que no final de cada case temos a palvra break seguida de ponto e vírgula, isso para a execução do switch, pois se já encontramos o caso não precisamos comparar ou outros casos. Repare também que temos a palavra default, esse é o código que será executado se nenhum dos casos forem verdadeiros, nesse exemplo se o usuário digitar qualquer coisa que não seja, "vermelho", "verde", "blue" ou "azul". Repare que temos case "azul": e case "blue": e depois o código que muda o fundo para azul. Isso é usado quando queremos executar o mesmo código para dois ou mais casos diferentes.

Loops

Muitas vezes precisamos repetir uma ação várias vezes. Por exemplo, imagine que temos um array com 10 elementos, cada um é o nome de um filme que já assisti. Se eu quero exibir o nome de todos filmes, será que preciso escrever isso 10 vezes, uma para cada filme? Posso até fazer isso, mas vai demorar um pouco e o código vai ficar um pouco grande. Agora imagine se fossem 100 filmes, ou 1000 filmes...

É para esse tipo de situação que os loops existem, vamos estudar 3 tipos de loop: while; do while; for.

while

Em português while significa enquanto. Esse loop executa um bloco de código enquanto uma certa condição for verdadeira. Veja o exemplo:

No arquivo app.js:

                
                    let numero = 6;
                    let chute;

                    while(numero != chute){
                        chute = prompt("Adivinhe o número:");
                    }

                    alert("Parabéns!");

Nesse exemplo criamos duas variáveis, em uma salvamos um número, a outra utilizamos para receber um valor digitado pelo usuário. No loop while, entre parenteses colocamos a condição que é o número ser diferente do chute do usuário. Ou seja o código entre chaves irá executar repetidamente enquanto o chute for diferente. Quando o usuário acertar o número, a condição numero != chute passa a ser falsa, então o loop para e o código segue para a próxima linha depois do loop, o alert.

Importante: é preciso ter certeza de que o loop tem um fim, ou seja, que a condição entre parenteses uma hora será falsa. Se isso nunca acontecer, entramos em um loop infinito, que roda até você fechar seu navegador, ou até o próprio navegador se sobrecarregar e parar sozinho ou exibir uma mensagem de erro.

do while

O loop do while é semelhante ao while. Porém tem 2 diferenças importantes:

1 - a sintaxe (forma de escrever) é um pouco diferente. Primeiro escrevemos a palavra do depois escrevemos o bloco de código entre chaves, e depois escrevemos a palavra while, seguida da condição entre parenteses.

2 - o código dentro do bloco sempre será executado da primeira vez, independentemente da condição ser verdadeira ou falsa. Só a partir da segunda vez é que a condição é avaliada. Veja o exemplo:

No arquivo app.js:

                
                    let contador = 10;

                    do {
                        alert(contador);
                        contador++;
                    } while(contador < 8);

Tente reescrever o exemplo acima usando while. Quantas vezes o contador é exibido na tela?

for

O último loop que estudaremos é o loop for, em portugês para. Diferente dos loops while e do while, o for recebe 3 valores entre parenteses, separados por ponto e vírgula.

for(início; condição; passo)

O primeiro valor é uma condição inicial, que geralmente é a declaração de uma variável com um valor inicial.

O segundo valor é a condição, que enquanto for verdadeira o loop continua.

O terceiro valor é algo que é executado ao final de cada execução do loop.

Veja o exemplo:

No arquivo app.js:

                
                    for(let i = 0; i < 5; i++){
                        alert(i);
                    }

Repare que o código dentro das chaves (o alert) é executado até que a condição i < 5 se torna falsa, ai então o loop para de executar.

A grande utilidade do for é fazer um loop dentro de um array, veja o exemplo:

No arquivo app.js:

                
                    let amigos = ["Fulano", "Ciclano", "Beltrano"];

                    for(let i = 0; i < amigos.length; i++){
                        alert("Olá " + amigos[i]);
                    }

Vamos analisar o código do exemplo anterior, pois ele é muito usado e importante.

Na condição inicial declaramos uma variável i e iniciamos ela com o valor 0. Lembre-se de que o primeiro elemento de um array é o elemento 0, e é acessado assim: amigos[0]. Portanto na primeira iteração do loop amigos[i] será igual a amigos[0].

A condição é i < amigos.length. Lembre-se que a propriedade length é a quantidade de elementos do array. E por quê temos i < amigos.length e não i <= amigos.length? Porque o indice do último elemento do array sempre será um a menos do que o tamanho do array, pois começamos a contagem em 0. Nesse exemplo o array amigos tem 3 elementos: amigos[0] é "Fulano"; amigos[1] é "Ciclano"; amigos[2] é "Beltrano";

O valor executado ao final de cada passo é i++, ou seja, ao final de cada passo adicionação 1 ao valor de i, assim podemos acessar o próximo valor do array.

Exercícios

Fila preferêncial

Escreva um script que pergunta ao usuário sua idade e se é uma pessoa com deficiência. Caso a pessoa tenha um deficiência ou tenha 60 anos ou mais, exiba uma mesagem "Senha preferêncial", caso contrário exiba uma mensagem "Senha comum".

Contratando alguém com experiência

Uma empresa quer contratar uma pessoa que seja maior de 18 anos e tenha mais de 6 meses de experiência com JavaScript. Faça um progama que pergunte ao usuário a sua idade e quantos meses de experiência em JavaScript ele tem. Se a pessoa satisfaz ambas condições exiba uma mensagem "Vamos marcar uma entrevista!", caso contrário exiba uma mensagem "Desculpe estamos procurando alguém mais experiente."

Bônus Noel

Uma empresa paga um bônus para seus funcionários no final do ano para que eles possam comprar presentes de Natal para seus filhos. Esse bônus depende do nṕumero de filhos. Quem têm 1 filho recebe R$50,00; Quem têm 2 filhos recebe R$90,00; Quem têm 3 filhos recebe R$130,00; Quem têm 4 filhos recebe R$165,00; Quem têm 5 filhos ou mais recebe R$40,00 mais R$30,00 vezes o número de filhos;

Escreva um script que pergunta ao usuário o número de filhos e exibe uma mensagem dizendo qual será o bônus dele.

Olá amigos!

Escreva um script que pede ao usuário que digite nomes de seus amigos, até ele deixar um em branco (string vazia). Depois disso exiba uma mensagem "Olá Fulano" para cada nome digitado.
